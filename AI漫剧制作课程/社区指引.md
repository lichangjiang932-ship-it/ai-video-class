### 答疑时间

| 类型 | 时间 | 方式 |
|------|------|------|
| **日常答疑** | 工作日 10:00-18:00 | 群内提问，助教回复 |
| **集中答疑** | 每周三 20:00-21:00 | 直播答疑（飞书/腾讯会议） |
| **紧急问题** | 随时 | 私聊助教 |

### 提问规范

好的提问方式：
```
【问题描述】使用可灵AI生成视频时，角色动作出现变形
【已尝试】1.调整了提示词 2.更换了参考图 3.降低了运动幅度
【工具版本】可灵AI v2.3
【截图/录屏】（附图）
```

避免的提问方式：
```
"视频生成不好用怎么办？"（太笼统，无法定位问题）
```

## 三、作品展示

### 作品分享格式

在群内分享作品时，建议使用以下格式：

```
【作品名称】《xxx》第x集
【题材类型】玄幻/都市/搞笑...
【使用工具】即梦AI + 可灵AI + 剪映
【制作时长】x小时/天
【作品链接】（平台链接或视频文件）
【自我评价】满意的地方 / 需要改进的地方
【求反馈】希望得到哪方面的建议
```

### 作品互评规则

1. **先说优点**：每个评价先说2个优点
2. **再提建议**：给出具体可操作的改进建议
3. **鼓励为主**：对初学者以鼓励为主，避免打击
4. **对事不对人**：评价作品，不评价个人

## 七、常见问题

**Q1：入群后需要改群昵称吗？**
A：建议改为"姓名-学习阶段"，如"张三-Week3"，方便交流。

**Q2：可以邀请朋友入群吗？**
A：可以，但需先联系助教确认。

**Q3：群里可以讨论课程以外的内容吗？**
A：可以适当讨论AI相关话题，但不要偏离太远。

**Q4：作业可以延期提交吗？**
A：可以，但建议提前和助教沟通。
