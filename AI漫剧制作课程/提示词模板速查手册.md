# 📝 AI漫剧提示词模板速查手册

> 本手册汇总课程中所有核心提示词模板，按使用场景分类，方便快速查找和复制使用。

## 一、剧本创作提示词

### 1.1 系统角色设定（一次性输入）

```
你是一位专业的AI漫剧编剧顾问，拥有以下设定：

【专业背景】
- 10年影视编剧经验，精通短视频叙事节奏
- 熟悉抖音、快手、B站、红果短剧的内容生态
- 擅长玄幻、都市、悬疑、甜宠四大热门题材
- 深度理解AI漫剧的制作流程和画面限制

【工作方式】
- 每集时长控制在1-3分钟（约300-800字剧本）
- 前3秒必须设置强钩子（冲突/悬念/反转）
- 每集结尾必须留悬念，驱动观众看下一集
- 对白要口语化、有记忆点
- 场景描述要可视化，方便AI生图/生视频

【输出格式】
- 使用标准剧本格式：场次号、场景、动作、对白
- 每个场次标注镜头建议（特写/中景/远景）
- 关键情绪节点用【】标注
- 每集结尾标注"钩子类型"

请确认你已理解以上设定，回复"编剧助手已就位"。
```

### 1.2 故事大纲生成

```
我要创作一部AI漫剧，请帮我生成故事大纲。

【基本设定】
- 题材类型：[玄幻/都市/悬疑/甜宠]
- 目标平台：[抖音/红果/B站]
- 总集数：[12/24/48/96集]
- 单集时长：[1分钟/2分钟/3分钟]
- 目标受众：[男性/女性/全龄]
- 核心爽点：[逆袭/复仇/甜宠/打怪升级]

【要求】
1. 故事核心设定（世界观/背景）
2. 主要角色设定（3-5个核心角色）
3. 故事主线（起承转合四个阶段）
4. 每集一句话概要
5. 爽点分布图（标注每集的情绪高低点）
```

### 1.3 单集剧本生成

```
请生成第[X]集的完整剧本。

【本集概要】：[粘贴大纲中本集的一句话概要]

【创作要求】
1. 剧本结构：场次号 + 场景描述 + 动作 + 对白 + 镜头建议
2. 节奏要求：第1场强钩子开场，最后1场大钩子结尾
3. AI适配要求：
   - 场景描述包含颜色、光线、氛围
   - 避免复杂多人同框（≤2-3人）
   - 动作描写简洁有力，适合图生视频
   - 每个场景附带AI生图提示词（英文）

总字数控制在600-800字。
```

## 二、图像生成提示词

### 2.1 角色立绘（即梦AI）

```
[角色名]，[年龄]岁，[性别]，[发型+发色]，[眼睛颜色]，
穿着[服装描述]，[表情]，[姿势]，
白色简洁背景，日系都市动漫风格，高质量角色立绘，全身像
```

**示例**：
```
林晓，25岁，男性，黑色短碎发，深棕色眼睛，
穿着深灰色连帽卫衣和黑色牛仔裤，平静的表情，站立姿势，
白色简洁背景，日系都市动漫风格，高质量角色立绘，全身像
```

### 2.2 场景母版图

```
[场景类型]内部/外部全景，[具体描述]，
[关键元素1]，[关键元素2]，[关键元素3]，
[光线描述]，[时间/氛围]，
日系都市动漫风格，高质量场景设计，无角色，纯场景画面，
16:9比例，精致的环境细节
```

### 2.3 完整镜头画面

```
[角色描述]，[动作/表情]，[在什么地方]，
[场景细节]，[光线]，[氛围]，
日系都市动漫风格，[镜头类型]，高质量
```

**镜头类型关键词**：
| 类型 | 中文 | 英文 |
|------|------|------|
| 大特写 | 极致特写 | extreme close-up |

| 全景 | 全身像 | full body shot |
| 远景 | 大场景 | wide shot |
| 俯视 | 俯视角度 | high angle |
| 仰视 | 仰视角度 | low angle |

## 三、视频生成提示词

### 3.1 可灵AI 文生视频公式

```
提示词 = 主体(描述) + 运动 + 场景(描述) + (镜头语言 + 光影 + 氛围)
```

**模板**：
```
[角色描述]，在[场景]中[动作描述]，
[光线描述]，[氛围]，电影级画面质感，1080p高清
```

**示例（打斗）**：
```
黑衣武士猛地向前跨步，右拳如闪电般挥出直击对方面门，
拳头划过空气带起一阵风声，身体重心前倾，
动作干脆利落，力量感十足，跟拍镜头，电影级动作场面
```

### 3.2 Seedance 2.0 参考视频驱动

**提示词模板**（动作由参考视频提供，提示词只描述静态元素）：
```
[角色名]在[场景]中[状态]，[服装描述]，
[环境细节]，[光线]，[氛围]，[风格]
```

**示例**：
```
林晓在雨中的城市街道上行走，身穿深灰色卫衣，
周围是霓虹灯招牌和湿润的柏油路面，
雨滴打在身上，冷色调的城市夜景灯光，
略带孤独的氛围，日系都市动漫风格
```

### 3.3 运动描述词库

| 类型 | 关键词 |
|------|--------|
| 打击 | punching, kicking, slashing, striking |
| 防御 | blocking, parrying, dodging |
| 空中 | leaping, aerial spin, dive attack |
| 能量 | casting a spell, energy blast |
| 行走 | walking, strolling, marching |
| 转身 | turning around, spinning |
| 坐下 | sitting down, kneeling |

## 四、配音提示词

### 4.1 火山引擎TTS 情感控制

```
# 基础格式
[情感标签]对白内容

# 示例
[开心]哈哈，终于等到你了！
[愤怒]你到底想怎么样！
[悲伤]对不起...我没能保护好你...
[紧张]快跑！他们追上来了！
```

### 4.2 剪映AI配音建议

- 在文字中加入标点控制停顿：`...` 表示停顿
- 用 `！` 表示强调
- 用 `？` 表示疑问语气
- 选择合适的AI声音（男声/女声、年轻/成熟）

## 七、提示词优化技巧

### 7.1 从模糊到精确

```
❌ 模糊：一个女孩在花园里
✅ 精确：一个16岁粉色双马尾少女，穿白色蕾丝连衣裙，
微笑站在盛开着红玫瑰的英式花园中，午后阳光，
日系清新动漫风格，中景正面，柔和自然光，超高质量
```

### 7.2 要素优先级

当字数受限时，按以下优先级保留：
```
必须保留：主体描述 + 动作表情 + 风格
强烈建议：环境 + 光线
可选省略：视角/景别
```

### 7.3 角色一致性锚定

每个镜头都包含固定的外貌描述：
```
【林晓-外貌锚定】
面部：25岁男性，棱角分明的脸型，深棕色瞳孔
发型：黑色短碎发，略微向右偏分
服装：深灰色连帽卫衣
```
