## 模块十一：剪辑后期

### 练习16：成片检查清单
完成一集成片后，逐项检查：

**前3秒**：
- [ ] 是否有视觉冲击？
- [ ] 是否有冲突呈现？
- [ ] 是否能抓住注意力？

**节奏**：
- [ ] 是否有张弛变化？
- [ ] 对话节奏是否自然？
- [ ] 有没有拖沓的地方？

**钩子**：
- [ ] 开头是否有回顾+悬念？
- [ ] 结尾是否留了钩子？
- [ ] 观众会想看下一集吗？

**技术**：
- [ ] 画面是否清晰？
- [ ] 音画是否同步？
- [ ] 字幕是否正确？
- [ ] 调色是否统一？
