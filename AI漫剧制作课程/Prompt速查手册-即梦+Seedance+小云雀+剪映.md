## 2. Seedance 2.0 · 视频生成

### 2.1 基础公式（适用 90% 场景）

```
首帧来源：◆即梦生成的关键帧 PNG
绑定角色档案：◆林晓_v1
镜头语言：◆中景特写
运镜：◆缓慢推进（dolly in）
动作描述：◆林晓抓头发3秒 → 抬头看屏幕2秒 → 烦躁地用力关上电脑3秒 → 闭眼仰头叹气2秒 → 重新睁眼5秒
情绪曲线：◆烦躁 → 短暂希望 → 失望 → 平静接受
环境互动：◆电脑屏幕红光在脸上闪烁，台灯保持稳定
氛围：◆孤独、绝望、转折前夕
时长：◆15s
画面比例：◆9:16
风格：◆电影感，浅景深
```

### 2.2 动作戏（追逐 / 打斗）

```
首帧：◆[角色 A 与 角色 B 对峙的关键帧]
角色档案：◆林晓_v1 + 张总_v1
镜头语言：◆主观镜头交叉中景
运镜：◆角色 A 视角 → 切换 角色 B 视角，每 3-4 秒切一次
动作描述：◆
  0-3s：林晓挥拳直击张总面部
  3-6s：张总后退一步，抹嘴角血迹
  6-9s：张总反击，抓林晓领口
  9-12s：两人扭打在地，镜头剧烈晃动
  12-15s：林晓压在上方，举拳停顿
情绪：◆愤怒爆发 → 失控 → 短暂清醒
镜头细节：◆手持感晃动，关键打击瞬间用 0.3s 慢动作
时长：◆15s
画面比例：◆16:9 横屏（动作戏建议横屏）
```

### 2.3 情绪特写（最容易出彩 + 最容易失败）

```
首帧：◆角色面部特写
角色档案：◆苏瑶_v1
镜头：◆面部大特写（仅入画从下巴到眉毛）
运镜：◆固定镜头，不动
动作：◆
  0-5s：苏瑶眼神空洞地盯着镜头
  5-8s：眼眶慢慢泛红
  8-11s：一滴眼泪从右眼角滑落
  11-15s：她闭上眼，眼泪继续流，嘴角微微抽动
情绪：◆压抑的崩溃
环境：◆背景全部虚化，只能看到模糊的黄色暖光
画面要求：◆**不要让角色说话**（Seedance 嘴型容易崩）
时长：◆15s
画面比例：◆9:16
```

### 2.4 风景空镜（用于场景切换）


## 3. 小云雀 · 短剧 Agent 输入模板

### 3.1 标准 docx 剧本格式

```
# AI漫剧剧本：◆《代码之外》第1集

## 角色表

### 林晓（男主）
- 年龄：◆25岁
- 性别：◆男
- 外貌描述：◆[完整复制即梦角色基准图的描述]
- 性格：◆内向倔强，技术宅，话少但执行力强

### 苏瑶（女主）
- 年龄：◆24岁
- 性别：◆女
- 外貌描述：◆[完整复制]
- 性格：◆温柔但有原则，洞察力强

## 场景表

### 场景1：林晓的出租屋·夜
[完整复制即梦场景母版描述]

### 场景2：互联网公司走廊·晨
[完整复制]

## 第1集

### 场景1·切片1（15秒）
[镜头：林晓的出租屋·夜·中景特写]
旁白：凌晨三点，林晓被项目折磨到崩溃。
（林晓抓头发，叹气，关上电脑）

### 场景1·切片2（15秒）
[镜头：林晓的出租屋·夜·特写]
（林晓望向窗外，自言自语）
林晓：再这样下去，我就废了……

### 场景2·切片3（15秒）
[镜头：互联网公司走廊·晨·全景]
旁白：第二天，林晓拖着疲惫的身体来到公司。
（林晓走过走廊，同事们窃窃私语，避开他的目光）
```

**关键技巧**：
- 每个切片严格 15 秒（约 30-50 字台词 + 1 个动作）
- 镜头描述放在 `[]` 里，小云雀会读取
- 角色对白前加角色名 + 冒号，旁白前加「旁白：」
- 动作放在 `()` 里
- **角色和场景描述完整复制即梦的，让小云雀也用一致风格**

### 3.2 小云雀生成视频时的设置

进入「故事蓝图」编辑器后，每个分镜检查 / 修改：

| 设置项 | 推荐值 | 备注 |
|--------|--------|------|
| 视频时长 | 15s | 与 Seedance 对齐 |
| 画面比例 | 9:16 / 16:9 | 与 Seedance 一致 |
| 画风 | 漫画 / 写实 | 与即梦保持一致 |
| 配音 | 关闭 | 在剪映里用克隆音色重配 |
| 字幕 | 关闭 | 在剪映里加 |
| 镜头时长 | 12-18s | 给剪映留切口余量 |

## 4. 剪映 · 后期模板包

### 4.1 调色预设（统一两条线画风）

新建调整图层 → 应用以下参数：

```
亮度：+5
对比度：+15
饱和度：+10
鲜艳度：+8
高光：-10
阴影：-5
白色：+5
黑色：-3
色温：+3（偏暖）
色调：-2（偏绿青）
锐化：+15
颗粒：+3（电影感）
LUT：电影感 - 暖调（剪映自带）
强度：60%
```

### 4.2 字幕模板

主字幕（旁白 / 角色对话）：

| 参数 | 值 |
|------|-----|
| 字体 | 站酷快乐体 / 思源黑体 CN Bold |
| 字号 | 80pt（竖屏）/ 48pt（横屏）|
| 颜色 | #FFFFFF（白）|
| 描边 | #000000（黑）4px |
| 阴影 | 黑色，距离 2，模糊 4 |
| 位置 | 底部居中，距底 100px |
| 入场动画 | 打字机效果 0.3s |
| 出场动画 | 渐隐 0.2s |

强调字幕（关键台词）：

| 参数 | 值 |
|------|-----|
| 字体 | 阿里巴巴普惠体 Heavy |
| 字号 | 120pt |
| 颜色 | #FFD700（金色）|
| 描边 | #FF0000（红）6px |
| 入场动画 | 弹跳放大 0.5s |
| 持续动效 | 轻微抖动循环 |

### 4.3 转场速查

| 切换类型 | 推荐转场 | 时长 |
|---------|---------|------|
| 同场景内切片 | 黑场闪白 | 0.3s |
| 切换场景 | 硬切 + 黑场 | 0.5s |
| 时间跳跃（"3 小时后"）| 闪白 + 字幕 | 1s |
| 回忆 / 闪回 | 模糊推出 | 0.6s |
| 高潮转折 | 镜头爆缩（zoom out） | 0.4s |
| 钩子结尾 | 慢动作 + 渐黑 | 1.5s |

## 6. 三个绝不要做的事

1. **绝不在视频里渲染中文** — 即梦/Seedance 的中文 99% 出乱码，**全部走剪映加字幕**
2. **绝不让 Seedance 处理对话戏** — 嘴型崩溃严重，对话戏用小云雀（自带配音 + 嘴型）
3. **绝不一次性生成超过 15 秒** — 当前所有 AI 视频工具，超过 15s 角色一致性必崩，强制按 15s 切片
