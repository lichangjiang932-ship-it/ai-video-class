## 核心工作流

```
即梦AI                          剪映
┌──────────────┐               ┌──────────────┐
│              │               │              │
│  ① 生成15s   │   首尾帧      │  ③ 拼接视频   │
│  电影质感视频 │ ──────────→  │  调色/字幕    │
│  （宠物拟人） │   提取        │  BGM/音效    │
│              │               │              │
│  ② 批量生成   │               │  ④ 导出成片   │
│  多个视频片段 │               │  竖屏9:16    │
│              │               │              │
└──────────────┘               └──────────────┘
```

**工作流说明**：
1. **即梦AI生成**：输入宠物拟人提示词，生成15s电影质感视频片段
2. **首尾帧提取**：从每个15s视频中提取关键帧，作为下一段的参考
3. **剪映拼接**：将多个视频片段拼接，添加配音、字幕、BGM
4. **成片输出**：竖屏9:16格式，适合抖音/B站/小红书分发

## 金猫搭档制作流程速览

```
Step 1: 写剧本
│  豆包："奶盖，你觉得人类为什么喜欢撸猫？"
│  奶盖："因为他们知道谁才是真正的主子。"
│
Step 2: 设计分镜
│  镜头1：豆包特写（温暖灯光）
│  镜头2：奶盖特写（冷色调）
│  镜头3：双人全景（播客间）
│
Step 3: 即梦AI生成图片
│  输入宠物拟人提示词 → 生成关键帧
│
Step 4: 即梦AI生成视频
│  关键帧 → 15s电影质感视频
│
Step 5: 提取首尾帧
│  每段视频的首帧/尾帧 → 下一段的参考
│
Step 6: 剪映拼接
│  多段视频拼接 + 配音 + 字幕 + BGM
│
Step 7: 成片输出
│  竖屏9:16 → 发布到抖音/B站/小红书
```
