## 第一部分：准备工作（5分钟）

### 1.1 注册登录

**网页版**（推荐，屏幕大好操作）：
1. 打开浏览器，地址栏输入：`xyq.jianying.com`
2. 点击右上角 **「登录」**
3. 选择 **「抖音登录」**（推荐）或 **「手机号登录」**
4. 授权登录

**手机APP**：
1. 打开应用商店（安卓）或 App Store（iOS）
2. 搜索 **「小云雀」**
3. 下载安装
4. 打开APP，用抖音/手机号登录

### 1.2 领取积分

登录后，系统每日赠送 **120积分**（约可生成12个智能视频）。
- 网页版：登录后自动领取
- APP版：打开APP自动领取

## 第二部分：上传剧本（3分钟）

### 2.1 进入短剧Agent

**网页版操作**：
1. 打开 `xyq.jianying.com`
2. 在首页找到 **「短剧Agent」** 入口（通常在功能列表的显著位置）
3. 点击进入

**APP操作**：
1. 打开小云雀APP
2. 在首页功能列表中找到 **「短剧Agent」**
3. 点击进入

### 2.2 上传剧本

1. 点击 **「上传剧本」** 或 **「新建短剧」** 按钮
2. 在文件选择器中找到刚才保存的 `代码之外-第1集.docx`
3. 选择文件，点击 **「打开」**
4. 等待系统解析（通常1-3分钟，会显示进度条）

### 2.3 确认需求设置

上传完成后，系统会弹出设置确认页面。按以下设置选择：

| 设置项 | 选择 | 说明 |
|--------|------|------|
| **画风** | **2D动漫** | 都市题材推荐2D |
| **叙事形式** | **沉浸式漫剧** | 角色对白驱动，无旁白解说 |
| **对白处理** | **保留原对白** | 先保留，后续可手动调整 |
| **目标平台** | **抖音** | 竖屏9:16 |

确认后点击 **「开始生成」** 或 **「确认」**。

## 第四部分：生成视频（30-60分钟）

### 4.1 直接生成（推荐新手）

**如果你不想编辑故事板，直接生成**：

1. 在故事板页面，找到 **「生成视频」** 按钮
2. 点击 **「生成视频」**
3. 系统开始按场次生成视频
4. 并行处理3-4场，耐心等待

**生成时间参考**：
- 8场戏 → 约30-60分钟
- 系统会显示每场的生成进度
- 可以关闭页面，后台继续生成

### 4.2 编辑后生成（进阶用户）

**如果你想调整某些镜头**：

1. 在故事板页面，点击你想修改的场景
2. 进入编辑模式，可以修改：
   - 镜头构图（拖动角色位置）
   - 画面描述（修改提示词）
   - 镜头类型（特写/中景/远景）
3. 修改完成后，点击 **「生成视频」**

### 4.3 等待生成完成

生成过程中，页面会显示：
```
场景1：生成中 ████████░░ 80%
场景2：已完成 ✅
场景3：生成中 ████░░░░░░ 40%
场景4：排队中 ⏳
场景5：排队中 ⏳
...
```

**最佳实践**：
> 如果集数多（如60集），建议白天准备好所有剧本，睡前统一点击生成，第二天早上起来收片。

### 4.4 下载视频

生成完成后：
1. 点击每场戏右侧的 **「下载」** 按钮
2. 视频以MP4格式保存到本地
3. 建议按场次命名：`场景1-出租屋深夜.mp4`

### 5.4 添加BGM

1. 点击剪映 **「音频」** → **「音乐」**
2. 搜索或浏览BGM：

| 场景 | BGM关键词 | 风格 |
|------|---------|------|
| 出租屋（场景1-2） | 轻柔钢琴/孤独/深夜 | 钢琴曲，慢节奏 |
| 公司（场景3-5） | 紧张/职场/压抑 | 低沉弦乐 |
| 天台（场景6-8） | 温暖/希望/转折 | 吉他曲，渐强 |

3. 将BGM拖入时间线音频轨道
4. 调整BGM音量（建议为配音音量的30%）

### 5.5 添加字幕

1. 点击剪映 **「文本」** → **「智能字幕」** → **「识别字幕」**
2. 剪映自动将配音转为文字字幕
3. 检查并修正识别错误
4. 调整字幕样式：
   - 字体：微软雅黑
   - 大小：中等
   - 颜色：白色
   - 位置：底部居中
   - 描边：黑色1px

### 5.6 调色

1. 点击时间线上的视频片段
2. 点击右侧 **「调节」** → **「滤镜」**
3. 选择滤镜：
   - 出租屋场景：偏冷色调（孤独感）
   - 公司场景：自然色调
   - 天台场景：暖色调（希望感）

### 5.7 添加AIGC标识

1. 点击 **「文本」** → **「新建文本」**
2. 输入：`本视频由AI生成`
3. 放在片尾最后1秒
4. 字体：白色，小号

### 5.8 导出

1. 点击右上角 **「导出」**
2. 设置：
   - 分辨率：**1080P**
   - 帧率：**30fps**
   - 格式：**MP4**
3. 点击 **「导出」**
4. 等待渲染完成
5. **导出后完整观看一遍**，确认无误

## 附录C：常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 剧本上传失败 | 确保是.docx格式，不是.txt或.pdf |
| 角色识别不全 | 在角色表中补充更详细的描述 |
| 角色形象不准确 | 补充参考实体信息（见第三部分3.3节） |
| 视频连贯性差 | 完善场景描述，补充参考实体信息 |
| 排队时间长 | 避开晚8-11点高峰期，或过夜生成 |
| 积分不够 | 每日登录领取120积分，或购买会员 |
| 画风不统一 | 同一批次使用相同画风设置 |
| 超过2人场景效果差 | 拆分为多个2人以内场景 |
| 对白太书面 | 用豆包优化对白网感（见附录D） |
| 视频画质不够 | 生成后用剪映"超分辨率"功能增强 |
