## 第一部分：准备工作（5分钟）

### 1.1 注册登录

**网页版**（推荐，屏幕大好操作）：
1. 打开浏览器，地址栏输入：`xyq.jianying.com`

3. 选择 **「抖音登录」**（推荐）或 **「手机号登录」**
4. 授权登录

1. 打开应用商店（安卓）或 App Store（iOS）
2. 搜索 **「小云雀」**
3. 下载安装
4. 打开APP，用抖音/手机号登录

登录后，系统每日赠送 **120积分**（约可生成12个智能视频）。

## 第二部分：上传剧本（3分钟）

### 2.1 进入短剧Agent

**网页版操作**：
1. 打开 `xyq.jianying.com`
2. 在首页找到 **「短剧Agent」** 入口（通常在功能列表的显著位置）
3. 点击进入

**APP操作**：
1. 打开小云雀APP
2. 在首页功能列表中找到 **「短剧Agent」**
3. 点击进入

### 2.2 上传剧本

1. 点击 **「上传剧本」** 或 **「新建短剧」** 按钮
2. 在文件选择器中找到刚才保存的 `代码之外-第1集.docx`
3. 选择文件，点击 **「打开」**
4. 等待系统解析（通常1-3分钟，会显示进度条）

### 2.3 确认需求设置

上传完成后，系统会弹出设置确认页面。按以下设置选择：

| 设置项 | 选择 | 说明 |
|--------|------|------|
| **画风** | **2D动漫** | 都市题材推荐2D |
| **叙事形式** | **沉浸式漫剧** | 角色对白驱动，无旁白解说 |
| **对白处理** | **保留原对白** | 先保留，后续可手动调整 |
| **目标平台** | **抖音** | 竖屏9:16 |

确认后点击 **「开始生成」** 或 **「确认」**。

## 第四部分：生成视频（30-60分钟）

### 4.1 直接生成（推荐新手）

**如果你不想编辑故事板，直接生成**：

1. 在故事板页面，找到 **「生成视频」** 按钮
2. 点击 **「生成视频」**
3. 系统开始按场次生成视频
4. 并行处理3-4场，耐心等待

**生成时间参考**：
- 8场戏 → 约30-60分钟
- 系统会显示每场的生成进度
- 可以关闭页面，后台继续生成

### 4.2 编辑后生成（进阶用户）

**如果你想调整某些镜头**：

1. 在故事板页面，点击你想修改的场景
2. 进入编辑模式，可以修改：
   - 镜头构图（拖动角色位置）
   - 画面描述（修改提示词）
   - 镜头类型（特写/中景/远景）
3. 修改完成后，点击 **「生成视频」**

### 4.3 等待生成完成

生成过程中，页面会显示：
```
场景1：生成中 ████████░░ 80%
场景2：已完成 ✅
场景3：生成中 ████░░░░░░ 40%
场景4：排队中 ⏳
场景5：排队中 ⏳
...
```

**最佳实践**：
> 如果集数多（如60集），建议白天准备好所有剧本，睡前统一点击生成，第二天早上起来收片。

### 4.4 下载视频

生成完成后：
1. 点击每场戏右侧的 **「下载」** 按钮
2. 视频以MP4格式保存到本地
3. 建议按场次命名：`场景1-出租屋深夜.mp4`

### 5.4 添加BGM

1. 点击剪映 **「音频」** → **「音乐」**
2. 搜索或浏览BGM：

| 场景 | BGM关键词 | 风格 |
|------|---------|------|
| 出租屋（场景1-2） | 轻柔钢琴/孤独/深夜 | 钢琴曲，慢节奏 |
| 公司（场景3-5） | 紧张/职场/压抑 | 低沉弦乐 |
| 天台（场景6-8） | 温暖/希望/转折 | 吉他曲，渐强 |

3. 将BGM拖入时间线音频轨道
4. 调整BGM音量（建议为配音音量的30%）

   - 字体：微软雅黑
   - 大小：中等
   - 颜色：白色
   - 位置：底部居中
   - 描边：黑色1px

### 5.6 调色

1. 点击时间线上的视频片段
2. 点击右侧 **「调节」** → **「滤镜」**
3. 选择滤镜：

   - 天台场景：暖色调（希望感）

1. 点击 **「文本」** → **「新建文本」**
2. 输入：`本视频由AI生成`
3. 放在片尾最后1秒
4. 字体：白色，小号

1. 点击右上角 **「导出」**
2. 设置：

   - 帧率：**30fps**
   - 格式：**MP4**

## 附录C：常见问题速查

|------|---------|
| 剧本上传失败 | 确保是.docx格式，不是.txt或.pdf |

   - 依次类推，直到场景8

### 8.5 添加BGM

7. **调整BGM音量**：
   - 点击BGM片段
   - 在右侧面板中找到 **「音量」** 滑块
   - 调整到25-35%（不要盖过配音）

### 8.6 添加字幕

1. 点击顶部菜单栏的 **「文本」** 选项卡
2. 在右侧面板中，找到 **「智能字幕」**
3. 点击 **「识别字幕」** 或 **「自动识别」**
4. 剪映会自动将所有配音转为文字字幕
5. **检查字幕**：
   - 逐条检查识别结果
   - 纠正识别错误的文字
   - 调整字幕的时间位置，使其与配音同步
6. **设置字幕样式**：
   - 点击字幕文本
   - 在右侧面板中设置：
     - 字体：**「微软雅黑」** 或 **「思源黑体」**
     - 大小：**「中等」**（约24-28号）
     - 颜色：**「白色」**
     - 描边：**「黑色」**，宽度1px
     - 位置：**底部居中**
     - 背景：无

### 8.7 调色

1. 点击时间线上的视频片段
2. 在右侧面板中，点击 **「调节」**
3. 可以手动调整：
   - **亮度**：出租屋场景调暗一点（-10），天台场景调亮一点（+10）
   - **对比度**：稍微增加（+5-10）
   - **饱和度**：天台场景可以稍微增加（+10），让夕阳更暖
   - **色温**：出租屋偏冷（-5），天台偏暖（+10）
4. 或者直接选择 **「滤镜」**：
   - 出租屋：选择偏冷色调的滤镜
   - 公司：选择自然色调的滤镜
   - 天台：选择暖色调的滤镜

### 8.8 添加转场

1. 在时间线上，找到两个片段之间的交界处
2. 将鼠标移到交界处，会出现一个 **「转场」** 图标
3. 点击图标，弹出转场效果库
4. 选择转场效果：
   - 场景切换（如出租屋→公司）：选择 **「淡入淡出」** 或 **「黑场过渡」**
   - 同一场景内（如场景1→场景2）：选择 **「无」** 或 **「叠化」**
5. 设置转场时长：建议 **0.5秒**

### 8.9 添加AIGC标识

1. 点击 **「文本」** → **「新建文本」**
2. 输入：`本视频由AI生成`
3. 设置样式：
   - 字体：微软雅黑
   - 大小：小号（约16号）
   - 颜色：白色
   - 位置：右下角
4. 将文本拖到时间线最后1秒的位置

### 8.10 导出

1. 点击右上角 **「导出」** 按钮
2. 在导出设置中：
   - 分辨率：选择 **「1080P」**（1920×1080）
   - 帧率：选择 **「30fps」**
   - 格式：选择 **「MP4」**
   - 编码：选择 **「H.264」**（兼容性最好）
3. 点击 **「导出」**
4. 等待渲染完成（通常1-3分钟）
5. **导出后必须完整观看一遍**，确认：
   - 画面是否清晰
   - 配音是否与画面同步
   - 字幕是否正确
   - BGM音量是否合适
   - 无明显错误

## 第九部分：小云雀其他功能实战

### 9.1 智能成片（快速出片）

**适用场景**：想快速出一个15-60秒的短视频，不需要详细剧本

**操作步骤**：
1. 打开小云雀 → **「智能成片」**
2. 输入一句话主题：

```
一个被裁员的程序员，在车库创业的故事
```

3. 选择视频风格：**「2D动漫」**
4. 选择时长：**「30秒」**
5. 点击 **「生成」**
6. 等待3-5分钟，系统自动生成视频
7. 预览，不满意可以重新生成

**优化技巧**：
- 主题描述越详细，生成效果越好
- 可以加上风格要求：「日系动漫风格」「暖色调」
- 可以加上情绪要求：「从压抑到希望的转变」

### 9.2 数字人口播（解说类内容）

**适用场景**：制作解说类漫剧、片头片尾介绍、课程讲解

**操作步骤**：
1. 打开小云雀 → **「数字人口播」**
2. 选择数字人形象：
   - 浏览200+可商用数字人形象
   - 选择一个符合你漫剧风格的形象
   - 男/女、年轻/成熟、正装/休闲
3. 输入口播文案：

```
大家好，欢迎收看《代码之外》。
这是一个关于程序员逆袭的故事。
林晓，一个25岁的程序员，被裁员后决定自己创业。
他会遇到什么样的困难？又会如何逆袭？
请关注我们的漫剧，每周更新。
```

4. 选择语言/方言：**「中文-普通话」**
5. 点击 **「生成」**
6. 等待生成完成
7. 下载视频，用作片头或片尾

### 9.3 爆款复刻（学习爆款结构）

**适用场景**：看到一个爆款视频，想学习它的结构并制作类似内容

**操作步骤**：
1. 在抖音上找到你想复刻的视频
2. 点击分享按钮 → **「复制链接」**
3. 打开小云雀 → **「爆款复刻」**
4. 粘贴视频链接
5. 系统自动分析视频的：
   - 镜头逻辑
   - 节奏结构
   - 画面风格
   - 音乐节奏
6. 点击 **「复刻」** 或 **「生成同款」**
7. 系统自动生成类似风格的视频

**注意**：
- 爆款复刻是学习结构，不是抄袭内容
- 建议复刻后大幅修改内容，创作自己的原创故事
- 可以复刻多个爆款，学习它们的共同规律

### 9.4 AI设计（封面/海报）

**适用场景**：制作漫剧封面、宣传海报、社交媒体图片

**操作步骤**：
1. 打开小云雀 → **「AI设计」**
2. 输入画面描述：

```
一个穿灰色卫衣的年轻男子，站在天台上望着黄昏的城市，
背后是橙红色的夕阳，旁边站着一个银白色长发的女孩，
日系动漫风格，电影级构图，暖色调
```

3. 选择尺寸：
   - 抖音封面：**「9:16」**
   - 微博/微信：**「1:1」** 或 **「16:9」**
4. 点击 **「生成」**
5. 从多个结果中选择最佳的
6. 下载使用

## 第十部分：批量生产工作流

### 10.1 一次准备多集剧本

**最高效的方式是一次准备好所有剧本，然后批量上传生成。**

**操作流程**：
1. 用DeepSeek一次性生成12集大纲（使用附录B的模板）
2. 用DeepSeek逐集生成完整剧本（每集约800字）
3. 将所有剧本合并到一个docx文件中
4. 上传到小云雀短剧Agent
5. 系统自动处理所有集数

### 10.2 批量docx文件结构

```
代码之外-完整剧本.docx

├── 角色表（放在最前面，所有集通用）
│   ├── 林晓

│   ├── 张总
│   └── 小胖
│
├── 第1集：困局
│   ├── 场景1：出租屋·深夜
│   ├── 场景2：出租屋·深夜（特写）

│   ├── 场景4：张总办公室·上午
│   ├── 场景5：公司工位·上午
```

### 10.3 过夜生成策略

**最佳实践**：
```
白天（9:00-18:00）：
  - 准备所有剧本
  - 上传到小云雀
  - 检查故事蓝图
  - 确认无误

晚上（22:00-23:00）：
  - 点击「生成视频」
  - 选择所有集数
  - 开始生成
  - 关闭电脑/手机

第二天早上（8:00+）：
  - 登录小云雀
  - 检查生成结果
  - 下载所有视频
  - 开始后期剪辑
```

### 10.4 团队协作分工

**如果是3-5人团队，推荐分工**：

| 角色 | 负责内容 | 工具 |
|------|---------|------|
| 编剧 | 写剧本、检查蓝图 | DeepSeek + 小云雀 |
| 制作 | 生成视频、质量检查 | 小云雀 |
| 后期 | 剪辑、配音、调色 | 剪映 |
| 运营 | 发布、数据分析、互动 | 抖音/B站 |

**每日流程**：
```
上午：编剧完成当日剧本 → 上传小云雀 → 检查蓝图
下午：制作生成视频 → 质量检查 → 不满意的重新生成
晚上：后期剪辑成片 → 运营发布
```
