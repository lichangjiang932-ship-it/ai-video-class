# 模块八：宠物拟人表演设计（第55-60课）

> **模块概述**：宠物拟人漫剧的灵魂在于"表演"——如何让金毛犬豆包和布偶猫奶盖做出自然的拟人动作（对嘴型、歪头、扒拉麦克风等），如何通过情绪表达让观众产生共鸣，如何用宠物独有的肢体语言增强角色魅力。本模块将从动作设计、情绪表达到提示词工程，系统解决宠物拟人表演的全部难题。

---

## 第55课：宠物拟人动作基础——让宠物"像人一样"动起来

### 学习目标

1. 理解宠物拟人动作设计的核心原则
2. 掌握即梦AI中生成宠物拟人动作的提示词技巧
3. 学会设计播客场景中的基础拟人动作
4. 能让豆包和奶盖在AI视频中做出自然的拟人动作

---

### 一、宠物拟人动作的设计原则

#### 1.1 什么是"好的"宠物拟人动作

好的宠物拟人动作需要同时满足两个条件：

| 条件 | 说明 | 示例 |
|------|------|------|
| 像人 | 动作符合人类的行为逻辑 | 坐在椅子上、端杯子、拍桌子 |
| 像宠物 | 保留宠物的物种特征 | 摇尾巴、竖耳朵、歪头 |

**核心平衡**：70%人类动作 + 30%宠物本能 = 自然的拟人动作

#### 1.2 宠物拟人动作的三个层次

| 层次 | 说明 | 示例 | 难度 |
|------|------|------|------|
| 基础层 | 简单的人类动作 | 坐、站、走路 | ★★ |
| 互动层 | 与物品/环境互动 | 端杯子、拍桌子、翻书 | ★★★ |
| 表演层 | 复杂的情绪表达 | 手舞足蹈、抱头痛哭 | ★★★★ |

#### 1.3 豆包和奶盖的动作风格差异

| 维度 | 豆包（金毛犬） | 奶盖（布偶猫） |
|------|---------------|---------------|
| 动作风格 | 幅度大、热情、外向 | 幅度小、优雅、内敛 |
| 标志动作 | 拍桌大笑、挥手 | 托腮、双手交叉 |
| 宠物本能 | 尾巴摇摆、耳朵竖起 | 尾巴轻摆、耳朵转动 |
| 身体语言 | 开放、舒展 | 收敛、端正 |

### 二、播客场景基础拟人动作

#### 2.1 坐姿动作

**标准坐姿**：
```
[角色描述]，端正地坐在播客桌前的椅子上，
背部挺直，双手自然放在桌面上，
目视前方，表情平静
```

**放松坐姿（豆包）**：
```
[豆包角色描述]，轻松地靠在椅背上，
一只脚搭在另一条腿上，一只手搭在椅子扶手上，
表情放松，嘴角带着微笑
```

**优雅坐姿（奶盖）**：
```
[奶盖角色描述]，优雅地坐在椅子上，
双腿并拢微微侧向一边，双手交叠放在膝盖上，
背部挺直，下巴微微抬起
```

#### 2.2 手部动作

**端杯子**：
```
[角色描述]，一只手端着咖啡杯，
手指自然弯曲握住杯身，另一只手搭在桌上，
嘴巴微微张开像在喝一口
```

**拍桌子**：
```
[角色描述]，一只手拍在桌面上，
手掌张开拍下的瞬间，表情兴奋或激动，
身体微微前倾
```

**举手说话**：
```
[角色描述]，右手举起，手掌张开做出强调的手势，
像是在说话时强调某个观点，
表情认真，嘴巴微张
```

**托腮**：
```
[角色描述]，一只手托着下巴，
手指自然弯曲贴着脸颊，
表情若有所思，眼睛微微眯起
```

#### 2.3 头部动作

**歪头**：
```
[角色描述]，头微微歪向一侧，
耳朵也跟着倾斜，表情困惑或好奇，
眼睛睁大看着某个方向
```

**点头**：
```
[角色描述]，头微微向下点，
像是在认真倾听并表示认同，
表情专注
```

**摇头**：
```
[角色描述]，头微微左右摇动，
像是在表示不同意或无奈，
表情略带苦笑
```

---

### 三、拟人动作的提示词技巧

#### 3.1 动作描述的层次结构

```
[身体姿态] + [四肢动作] + [头部动作] + [面部表情] + [宠物特征]
```

**完整示例**：
```
[豆包角色描述]，坐在播客桌前，
身体微微前倾（身体姿态），
右手举起做出强调的手势（四肢动作），
头微微歪向一侧（头部动作），
表情兴奋，嘴巴大张（面部表情），
耳朵竖起，尾巴在身后摇摆（宠物特征），
暖色调灯光，电影质感，竖屏9:16
```

#### 3.2 动作描述的常见陷阱

| 陷阱 | 问题 | 正确做法 |
|------|------|---------|
| 太笼统 | "豆包在动" | 具体描述每个身体部位 |
| 太复杂 | 同时做3个以上动作 | 每个镜头只做1-2个动作 |
| 忽略宠物特征 | 只描述人类动作 | 始终加入耳朵、尾巴等宠物特征 |
| 动作与情绪不匹配 | 悲伤时拍桌 | 动作要与情绪一致 |

---

### 四、常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 动作僵硬不自然 | 描述太死板 | 加入"自然""放松""微微"等修饰词 |
| 手/爪子变形 | AI对手部处理弱 | 避免手部特写，用物品遮挡 |
| 宠物特征消失 | 只描述了人类动作 | 始终加入耳朵、尾巴的描述 |
| 动作幅度过大 | 运动幅度设置太高 | 降低运动幅度到"低-中" |

---

### 五、课后练习

3. **练习3**：生成3种头部动作（歪头、点头、摇头）
4. **练习4**：尝试组合"身体+手+头+表情"的完整动作
5. **思考题**：如何在AI视频中让宠物拟人动作看起来不"假"？

## 第56课：对嘴型与说话表演——让宠物"开口说话"

### 学习目标

1. 理解宠物拟人角色说话表演的设计方法
2. 掌握用即梦AI生成说话动作的技巧
3. 学会设计不同情绪下的说话表演
4. 能让豆包和奶盖的说话表演自然生动

---

### 一、宠物拟人说话的视觉设计

#### 1.1 嘴型变化设计

宠物拟人角色的嘴型设计需要兼顾"宠物嘴型"和"人类说话"：

| 嘴型状态 | 豆包（金毛犬） | 奶盖（布偶猫） |
|---------|---------------|---------------|
| 闭嘴 | 嘴巴自然闭合，嘴唇覆盖牙齿 | 小巧的嘴巴闭合 |
| 微张 | 嘴巴微微张开，露出一点牙齿 | 嘴巴微微张开 |
| 说话 | 嘴巴张开，舌头偶尔可见 | 嘴巴张合，幅度较小 |
| 大笑 | 嘴巴大张，舌头伸出，露齿 | 嘴巴张开，眼睛眯起 |

#### 1.2 说话时的身体配合

| 情绪 | 说话动作 | 身体配合 |
|------|---------|---------|
| 兴奋讲述 | 嘴巴快速张合 | 身体前倾，手势丰富 |
| 平静叙述 | 嘴巴缓慢张合 | 身体端正，手势少 |
| 愤怒反驳 | 嘴巴有力张合 | 身体前倾，手拍桌 |
| 温柔安慰 | 嘴巴轻微张合 | 身体微微侧向对方 |

---

### 二、说话表演的提示词设计

#### 2.1 标准说话模板

```
[角色描述]，坐在播客麦克风前，
嘴巴微微张开像在说话，
[情绪描述的表情]，
[身体姿态和手势]，
[宠物特征：耳朵、尾巴]，
[场景和灯光]，电影质感，竖屏9:16
```

#### 2.2 不同情绪的说话表演

**兴奋说话（豆包）**：
```
[豆包角色描述]，坐在播客麦克风前，
嘴巴大张像在兴奋地说话，表情热情洋溢，
身体微微前倾，右手举起做手势强调，
耳朵竖起，尾巴在身后快速摇摆，
暖色调灯光，电影质感，竖屏9:16
```

**傲娇说话（奶盖）**：
```
[奶盖角色描述]，坐在播客桌对面，
嘴巴微微张开像在说话，嘴角带着一丝傲娇的微笑，
微微仰头，一只手优雅地端着咖啡杯，
耳朵微微转动，尾巴轻轻搭在椅子上，
暖色调灯光，电影质感，竖屏9:16
```

**认真说话（豆包）**：
```
[豆包角色描述]，坐在播客麦克风前，
嘴巴缓慢张合像在认真地讲述，
表情专注，眉头微皱，双手交叠放在桌上，
耳朵自然下垂，尾巴安静地垂在身后，
暖色调灯光，电影质感，竖屏9:16
```

**温柔说话（奶盖）**：
```
[奶盖角色描述]，坐在播客桌对面，
嘴巴轻微张合像在温柔地说话，
表情柔和，眼神温暖，双手捧着咖啡杯，
耳朵微微前倾，尾巴轻轻摆动，
暖色调灯光，电影质感，竖屏9:16
```

### 三、说话与倾听的配合

#### 3.1 对话节奏设计

```
说话者A（5秒）→ 倾听者B反应（2秒）→ 说话者B（5秒）→ 倾听者A反应（2秒）
```

#### 3.2 倾听者的表演设计

| 倾听反应 | 画面 | 适用场景 |
|---------|------|---------|
| 认真倾听 | 眼睛注视对方，微微点头 | 正常对话 |
| 惊讶倾听 | 眼睛睁大，嘴巴微张 | 听到意外消息 |
| 思考倾听 | 单手托腮，眼睛微眯 | 听到复杂话题 |
| 傲娇倾听 | 双手交叉，微微仰头 | 听到夸奖 |
| 搞笑倾听 | 捂嘴笑，眼睛弯起 | 听到搞笑的事 |

---

### 四、AI视频中的说话表演技巧

#### 4.1 即梦AI视频中模拟说话

即梦AI生成的视频本身没有语音，但可以通过以下方式模拟说话效果：

**方法一：嘴部动态描述**
```
嘴巴在动像在说话，嘴巴张合的幅度适中
```

**方法二：配合手势**
```
嘴巴在动像在说话，同时右手举起做手势
```

**方法三：配合表情变化**
```
嘴巴在动像在说话，表情从平静逐渐变为兴奋
```

#### 4.2 后期配音对齐

在剪映中添加AI配音时，需要注意：
- 配音的语速与视频中嘴巴张合的频率匹配
- 配音的起始时间与嘴巴开始动的时间对齐
- 如果视频中嘴巴不动，配音时角色应该在"倾听"状态

---

### 五、常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 嘴巴不动 | 提示词中没有描述嘴部 | 明确写"嘴巴在动像在说话" |
| 嘴巴变形 | 运动幅度太大 | 降低运动幅度 |
| 说话时身体僵硬 | 只描述了嘴巴 | 同时描述身体和手势 |
| 说话和倾听分不清 | 没有区分两种状态 | 明确写"在说话"或"在倾听" |

## 第57课：歪头杀与萌系动作——宠物独有的魅力表演

### 学习目标

1. 理解宠物独有的萌系动作在漫剧中的价值
2. 掌握"歪头杀"等经典宠物动作的生成方法
3. 学会设计宠物独有的魅力动作
4. 能用萌系动作增强豆包和奶盖的角色魅力

---

### 一、宠物独有的萌系动作

#### 1.1 经典萌系动作清单

| 动作 | 描述 | 魅力指数 | 使用场景 |
|------|------|---------|---------|
| 歪头杀 | 头歪向一侧，耳朵跟着倾斜 | ★★★★★ | 困惑、好奇、撒娇 |
| 耳朵飞机耳 | 两只耳朵向后压平 | ★★★★ | 害怕、紧张、委屈 |
| 瞳孔放大 | 眼睛突然变大变圆 | ★★★★★ | 惊讶、看到喜欢的东西 |
| 舔爪子/手 | 用手/爪子舔一下 | ★★★ | 尴尬、紧张 |
| 甩头 | 快速甩动头部 | ★★★ | 挣扎、摆脱 |
| 蜷缩 | 身体蜷缩成一团 | ★★★★ | 害冷、害怕、撒娇 |
| 伸懒腰 | 双手举起伸展身体 | ★★★ | 刚睡醒、放松 |
| 打哈欠 | 嘴巴大张打哈欠 | ★★★ | 困倦、无聊 |

#### 1.2 豆包专属萌系动作

| 动作 | 描述 | 使用场景 |
|------|------|---------|
| 尾巴螺旋桨 | 尾巴快速摇摆成圆形 | 极度开心 |
| 大舌头笑 | 嘴巴大张，舌头伸出 | 大笑、傻乐 |
| 前爪搭桌 | 两只前爪搭在桌面上 | 期待、撒娇 |
| 耳朵一竖一垂 | 一只耳朵竖起一只耳朵下垂 | 困惑、思考 |

#### 1.3 奶盖专属萌系动作

| 动作 | 描述 | 使用场景 |
|------|------|---------|
| 尾巴缠绕 | 尾巴缠绕在椅子腿上 | 紧张、害羞 |
| 猫式伸展 | 身体向前伸展，臀部翘起 | 放松、伸懒腰 |
| 耳朵转动 | 耳朵向不同方向转动 | 警觉、好奇 |
| 用尾巴遮脸 | 尾巴举起来遮住半边脸 | 害羞、尴尬 |

---

### 二、歪头杀的完美生成

#### 2.1 歪头杀的视觉要素

| 要素 | 描述 |
|------|------|
| 头部角度 | 歪向一侧约15-30度 |
| 耳朵 | 随头部倾斜，一只高一只低 |
| 眼睛 | 睁大，带着好奇/困惑 |
| 嘴巴 | 微微张开或闭合 |
| 身体 | 保持不动或微微侧倾 |

#### 2.2 豆包歪头杀提示词

```
[豆包角色描述]，坐在播客桌前，
头微微歪向右侧约20度，
右耳随之抬高，左耳自然下垂，
眼睛睁大带着好奇的表情，嘴巴微微张开，
像在听到什么不理解的事情，
暖色调灯光，电影质感，竖屏9:16
```

#### 2.3 奶盖歪头杀提示词

```
[奶盖角色描述]，坐在播客桌对面，
头微微歪向左侧，
左耳随之倾斜，右耳保持竖立，
蓝色大眼睛睁大，表情困惑又可爱，
嘴角微微下撇，像在思考什么，
暖色调灯光，电影质感，竖屏9:16
```

---

### 三、萌系动作在漫剧中的应用

#### 3.1 用萌系动作增强情感表达

| 情感 | 传统表达 | 萌系表达 | 效果提升 |
|------|---------|---------|---------|
| 困惑 | 皱眉思考 | 歪头+耳朵一竖一垂 | 更可爱、更有趣 |
| 害怕 | 身体后缩 | 飞机耳+蜷缩 | 更真实、更有共鸣 |
| 开心 | 微笑 | 尾巴摇摆+瞳孔放大 | 更生动、更有感染力 |
| 尴尬 | 挠头 | 舔爪子+眼神游移 | 更有宠物特色 |
| 撒娇 | 说话变软 | 前爪搭桌+歪头 | 更萌、更有魅力 |

#### 3.2 萌系动作的节奏控制

| 动作类型 | 时长 | 运动幅度 | 说明 |
|---------|------|---------|------|
| 歪头杀 | 3秒 | 低 | 头部微动即可 |
| 尾巴摇摆 | 5秒 | 中 | 配合表情变化 |
| 瞳孔放大 | 2秒 | 低 | 瞬间反应 |
| 伸懒腰 | 5秒 | 中 | 全身伸展 |
| 蜷缩 | 3-5秒 | 低-中 | 身体收缩 |

---

### 四、常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 歪头角度太大 | 描述角度太夸张 | 写"微微歪向一侧"，不写具体角度 |
| 耳朵动作不自然 | AI对耳朵控制弱 | 用参考图法，以有耳朵动作的图为参考 |
| 萌系动作太假 | 动作与情绪不匹配 | 确保萌系动作有合理的情绪触发 |
| 宠物特征消失 | 只描述了人类动作 | 始终加入耳朵、尾巴的描述 |

## 第58课：情绪表达设计——让观众感同身受

### 一、宠物拟人情绪表达的特殊性

#### 1.1 与人类角色的区别

| 维度 | 人类角色 | 宠物拟人角色 |
|------|---------|------------|
| 面部表情 | 靠面部肌肉 | 靠面部+耳朵+尾巴 |
| 肢体语言 | 手势、姿态 | 手势+宠物本能动作 |
| 情绪层次 | 主要靠表情 | 表情+毛发状态+瞳孔变化 |
| 表达幅度 | 可以很微妙 | 可以更夸张（因为有宠物特征加成） |

#### 1.2 宠物拟人的情绪表达公式

```
情绪 = 面部表情 + 耳朵状态 + 尾巴状态 + 身体姿态 + 毛发状态
```

### 二、六种核心情绪设计

#### 2.1 开心

**豆包的开心**：
```
面部：嘴巴大张笑，眼睛弯成月牙，露出牙齿
耳朵：完全竖起
尾巴：快速摇摆
身体：身体舒展，可能拍桌或举手
毛发：蓬松有光泽
```

**奶盖的开心**：
```
面部：嘴角上扬，眼睛眯起，可能捂嘴笑
耳朵：微微前倾
尾巴：轻轻摆动
身体：身体微微前倾，双手可能捧脸
毛发：柔软蓬松
```

**提示词（豆包开心）**：
```
[豆包角色描述]，坐在播客桌前，
嘴巴大张开心地笑，眼睛弯成月牙，露出整齐的牙齿，
耳朵完全竖起，尾巴在身后快速摇摆，
身体微微后仰，右手拍着桌面，
暖色调灯光，电影质感，竖屏9:16
```

#### 2.2 愤怒

**豆包的愤怒**：
```
面部：眉头紧锁，龇牙，眼睛瞪大
耳朵：向后压平（飞机耳）
尾巴：僵硬竖起或快速甩动
身体：身体前倾，双手握拳
毛发：微微炸起（特别是脖子周围的毛）
```

**奶盖的愤怒**：
```
面部：瞳孔变细，嘴唇紧抿，眉毛上挑
耳朵：完全向后压平
尾巴：膨大发炸
身体：弓起背部，双手交叉或拍桌
毛发：全身微微炸起
```

**提示词（奶盖愤怒）**：
```
[奶盖角色描述]，坐在播客桌对面，
瞳孔变细，嘴唇紧抿，眉毛上挑带着怒意，
耳朵完全向后压平，尾巴膨大发炸，
弓起背部，一只手拍在桌面上，
暖色调灯光，电影质感，竖屏9:16
```

#### 2.3 悲伤

**豆包的悲伤**：
```
面部：眼睛低垂，眼角下垂，嘴角下弯
耳朵：完全耷拉下来
尾巴：无力下垂
身体：身体蜷缩，肩膀下垂
毛发：失去光泽，微微凌乱
```

**奶盖的悲伤**：
```
面部：眼含泪光，嘴角微微下撇
耳朵：后压，微微颤抖
尾巴：缠绕在自己身上
身体：低头，双手抱住自己
毛发：微微凌乱
```

#### 2.4 惊讶

**豆包的惊讶**：
```
面部：眼睛瞪到最大，嘴巴O型张开
耳朵：突然竖起
尾巴：瞬间僵直
身体：身体微微后仰
毛发：微微炸起
```

**奶盖的惊讶**：
```
面部：眼睛圆睁，嘴巴微张
耳朵：完全前倾，像雷达一样
尾巴：瞬间膨大
身体：身体后仰，一只手捂嘴
毛发：微微炸起
```

#### 2.5 思考

**豆包的思考**：
```
面部：眉头微皱，眼睛看向上方或远方
耳朵：一只竖起一只下垂（不对称）
尾巴：缓慢摆动
身体：一只手托下巴
毛发：正常状态
```

**奶盖的思考**：
```
面部：眼睛微眯，嘴角微微抿起
耳朵：微微转动，像在接收信息
尾巴：缓慢摆动
身体：单手托腮，身体微微侧倾
毛发：正常状态
```

#### 2.6 尴尬

**豆包的尴尬**：
```
面部：嘴角僵硬地笑，眼神游移
耳朵：一只竖一只垂
尾巴：僵硬地停住
身体：一只手挠头
毛发：正常状态
```

**奶盖的尴尬**：
```

耳朵：转向两侧
尾巴：缠绕在椅子腿上
身体：双手不知道放哪里
毛发：正常状态
```

### 三、情绪的层次与递进

#### 3.1 情绪弧线设计

一段对话中的情绪不是一成不变的，而是有起伏的：

```
平静 → 好奇 → 兴奋 → 惊讶 → 思考 → 感悟 → 开心
```

#### 3.2 情绪过渡的画面设计

| 情绪过渡 | 画面变化 | 时长 |
|---------|---------|------|
| 平静→好奇 | 眉毛微挑，耳朵前倾 | 2秒 |
| 好奇→兴奋 | 眼睛发亮，身体前倾 | 3秒 |
| 兴奋→惊讶 | 眼睛瞪大，嘴巴张开 | 1秒 |
| 惊讶→思考 | 眼睛微眯，单手托腮 | 3秒 |
| 思考→开心 | 嘴角上扬，眼睛弯起 | 2秒 |

### 四、常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 情绪不明显 | 表情描述太笼统 | 用具体的面部肌肉和宠物特征描述 |
| 情绪太夸张 | 描述太强烈 | 用"微微""略微"等修饰词 |
| 宠物特征与情绪不匹配 | 耳朵/尾巴状态不对 | 参考本课的情绪对照表 |
| 情绪过渡太突然 | 缺乏中间状态 | 设计情绪弧线，有过渡 |
