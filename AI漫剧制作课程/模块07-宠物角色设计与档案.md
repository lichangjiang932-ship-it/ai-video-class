### 一、什么是角色基准图

#### 1.1 定义

角色基准图是角色的"标准照"——一张最能代表角色外貌特征的高质量图片。后续所有场景、表情、动作的生成都以基准图为参考，确保角色始终如一。

#### 1.2 基准图的质量要求

| 要求 | 说明 |
|------|------|
| 分辨率高 | 至少1024×1920（竖屏） |
| 角色清晰 | 五官、配饰、服装细节清晰可辨 |
| 背景简洁 | 纯色或简单背景，不干扰角色 |
| 光线均匀 | 无强烈阴影，面部光线充足 |
| 姿态标准 | 站立或坐姿端正，展示全身特征 |
| 风格明确 | 确定角色的画风（写实动漫/Q版/水彩等） |

### 二、宠物拟人角色的视觉标识设计

#### 2.1 什么是视觉标识

视觉标识是让观众**一眼认出角色**的关键特征。对于宠物拟人角色，视觉标识包括：

| 标识类型 | 豆包（金毛犬） | 奶盖（布偶猫） |
|---------|---------------|---------------|
| 物种特征 | 金毛犬，金色蓬松毛发 | 布偶猫，蓝眼睛，重点色 |
| 标志配饰 | 黑框眼镜 | 粉色蝴蝶结领结 |
| 服装风格 | 蓝色卫衣+卡其色工装裤 | 奶油色针织衫 |
| 体型特征 | 健壮，耳朵自然下垂 | 娇小圆润，尾巴蓬松 |
| 性格外化 | 温暖笑容，大眼睛 | 微微仰头，傲娇表情 |
| 色彩系统 | 暖金+蓝色 | 米白+粉色 |

#### 2.2 视觉标识设计原则

**原则一：差异化**

两个角色的视觉标识必须有明显差异，不能混淆。

| 维度 | 豆包 | 奶盖 | 差异 |
|------|------|------|------|
| 主色调 | 金色+蓝色 | 米白+粉色 | 冷暖色差异 |
| 配饰 | 黑框眼镜 | 粉色蝴蝶结 | 方形vs圆形 |
| 体型 | 健壮 | 娇小 | 大小差异 |
| 耳朵 | 下垂 | 竖立 | 形态差异 |

**原则二：简洁性**

标识不能太复杂，要让人一眼记住。

**原则三：跨画风稳定性**

标识在任何画风下都应清晰可辨。比如豆包的眼镜在写实动漫、水彩、Q版中都应该存在。

### 三、豆包基准图创建

#### 3.1 豆包角色设定文档

```
角色名：豆包
物种：金毛犬（拟人化）
年龄设定：25岁左右（人类等效）
性别：男
体型：健壮，中等身高

外貌特征：
- 毛发：金色蓬松长毛，耳朵处最蓬松
- 耳朵：自然下垂，金黄色
- 眼睛：大而温暖，深棕色
- 鼻子：黑色湿润
- 尾巴：金色蓬松，经常摇摆

标志配饰：
- 黑框方形眼镜
- 蓝色卫衣（略宽松）
- 卡其色工装裤
- 白色运动鞋

性格外化：
- 标志表情：温暖的大笑
- 肢体语言：手势丰富，喜欢拍桌
- 姿态习惯：身体前倾，表示认真
```

### 五、基准图质量检查

#### 5.1 检查清单

| 检查项 | 豆包 | 奶盖 |
|--------|------|------|
| 物种特征正确？ | 金毛犬 | 布偶猫 |
| 毛发颜色正确？ | 金色 | 重点色（深+浅） |
| 配饰完整？ | 眼镜+蓝色卫衣 | 蝴蝶结+奶油针织衫 |
| 体型正确？ | 健壮 | 娇小 |
| 眼睛正确？ | 深棕色 | 蓝色 |
| 背景简洁？ | 纯白 | 纯白 |
| 分辨率足够？ | 1024×1920+ | 1024×1920+ |
| 风格统一？ | 写实动漫 | 写实动漫 |

#### 5.2 两个角色的差异化检查

将豆包和奶盖的基准图并排放置，检查：
- [ ] 能一眼区分两个角色？
- [ ] 配色系统完全不同？（金色+蓝色 vs 米白+粉色）
- [ ] 体型差异明显？
- [ ] 配饰不会混淆？（眼镜 vs 蝴蝶结）

### 六、常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 金毛犬变成拉布拉多 | 品种描述不精确 | 增加"金色蓬松长毛""耳朵下垂"等词 |
| 布偶猫重点色不明显 | 毛色描述不足 | 明确写"脸和耳朵深棕色，身体浅米色" |
| 眼镜总是消失 | 配饰权重不够 | 将"戴着黑框眼镜"放在描述靠前位置 |
| 蝴蝶结变成其他饰品 | 描述不够精确 | 明确写"粉色蝴蝶结领结，系在脖子正前方" |
| 两个角色太像 | 物种特征不够突出 | 加强物种特征词，增加差异化配饰 |

## 第50课：角色档案建立——从图片到完整档案

### 学习目标

1. 理解角色档案的完整结构
2. 掌握为宠物拟人角色建立多维度档案的方法
3. 学会将基准图、性格、动作特征整合为统一档案
4. 能为豆包和奶盖建立可复用的角色档案文件

---

### 一、角色档案的完整结构

#### 1.1 档案包含的维度

```
角色档案
├── 基础信息（名字、物种、年龄、性别）
├── 视觉特征（外貌、配饰、服装、色彩）
├── 性格设定（性格、说话风格、口头禅）
├── 动作特征（标志动作、习惯姿态、情绪表达方式）
├── 参考图集（多角度基准图、表情图、动作图）
├── 提示词模板（标准化的生成提示词）
└── 生成参数（种子值、参考强度、画风关键词）
```

#### 1.2 为什么需要完整档案

| 没有档案 | 有档案 |
|---------|--------|
| 每次重新描述角色 | 复制粘贴标准化描述 |
| 角色特征时有时无 | 特征始终一致 |
| 性格表达混乱 | 性格表达统一 |
| 换人制作时风格断层 | 任何人制作风格统一 |

# 角色档案：豆包

## 基础信息
- 角色名：豆包
- 物种：金毛犬（拟人化）
- 年龄设定：25岁（人类等效）
- 性别：男
- 身份：播客主持人
- 身高：中等偏高（比奶盖高一个头）

## 视觉特征
### 毛发
- 颜色：金色
- 质感：蓬松长毛，耳朵处最蓬松
- 特点：阳光下有光泽

### 面部
- 眼睛：大而温暖，深棕色瞳孔
- 耳朵：自然下垂，金黄色
- 鼻子：黑色湿润
- 嘴巴：微笑时露出整齐的牙齿

### 标志配饰
- 黑框方形眼镜（始终佩戴）
- 蓝色卫衣（略宽松，胸前可能有小图案）
- 卡其色工装裤（多口袋设计）
- 白色运动鞋

### 色彩系统
- 主色：金色（毛发）
- 辅色：蓝色（卫衣）
- 点缀：卡其色（裤子）、白色（鞋子）
- 整体色调：暖色系

## 性格设定
### 核心性格
- 温暖、乐观、热情
- 偶尔犯傻，但很真诚
- 喜欢开玩笑，是气氛担当
- 对朋友很忠诚

### 说话风格
- 语速偏快，语调上扬
- 喜欢用感叹句
- 偶尔冒出金句

### 情绪特点
- 开心时：尾巴摇摆，眼睛弯成月牙

- 难过时：耳朵耷拉，声音变小

## 动作特征
### 标志动作
- 拍桌大笑（搞笑时刻）
- 举手强调（认真讲述时）
- 挠头（尴尬/困惑时）
- 摊手（无奈时）

### 习惯姿态
- 坐着时身体微微前倾（表示认真）
- 说话时手势丰富
- 倾听时单手托腮
- 放松时靠在椅背上

### 肢体语言
- 兴奋：前爪举起，身体前倾
- 紧张：爪子不停挠桌角
- 开心：尾巴摇摆，眼睛弯起
- 思考：一只爪子托下巴，看向上方

- 耳朵：三角形竖立，深棕色

- 粉色蝴蝶结领结（系在脖子正前方，始终佩戴）

### 核心性格
- 傲娇但内心善良
- 聪明、有主见
- 偶尔毒舌，但出发点是好的
- 对豆包很包容

### 说话风格
- 语速适中，语调平稳
- 喜欢用反问句
- 偶尔冒出犀利点评
- 口头禅："哦？""你确定？""真是的...""笨蛋。"

### 情绪特点
- 傲娇时：微微仰头，嘴角上扬
- 开心时：眯眼微笑，尾巴轻摆
- 惊讶时：眼睛圆睁，耳朵前倾
- 难过时：低头，耳朵后压

## 动作特征
### 标志动作
- 双手交叉抱胸（傲娇/反驳时）
- 托腮倾听（认真时）
- 端咖啡杯（放松时）
- 微微仰头（自信/傲娇时）

### 习惯姿态
- 坐着时身体端正（优雅）
- 说话时手势不多但精准
- 倾听时单手托腮
- 放松时蜷在椅子上

- 傲娇：双手交叉，微微仰头
- 开心：眼睛眯起，嘴角上扬
- 惊讶：身体后仰，手捂嘴

## 第53课：跨画风角色识别——任何画风都能认出你

### 学习目标

1. 理解跨画风角色识别的核心挑战
2. 掌握在不同画风中保持角色可识别性的方法
3. 学会为豆包和奶盖设计"跨画风标识系统"
4. 能让角色在写实、水彩、Q版等画风中都被认出

---

### 一、跨画风识别的核心挑战

#### 1.1 不同画风的信息损失

| 画风 | 保留的特征 | 容易丢失的特征 |
|------|-----------|---------------|
| 写实动漫 | 全部特征 | 无 |
| 水彩插画 | 色彩和轮廓 | 细节（眼镜框、蝴蝶结细节） |
| Q版卡通 | 色彩和大特征 | 体型比例、细节 |
| 像素风 | 色彩块 | 几乎所有细节 |
| 国风水墨 | 轮廓和意境 | 色彩、细节 |

#### 1.2 宠物拟人角色的特殊挑战

| 挑战 | 说明 |
|------|------|
| 物种差异模糊 | Q版中金毛犬和拉布拉多可能一样 |
| 配饰简化 | 低细节画风中配饰可能消失 |
| 拟人化程度变化 | 不同画风的拟人化程度不同 |

---

### 二、跨画风标识系统设计

#### 2.1 核心标识（任何画风都必须保留）

| 标识 | 豆包 | 奶盖 | 保留优先级 |
|------|------|------|-----------|
| 主色调 | 金色+蓝色 | 米白+粉色 | ★★★★★ |
| 标志配饰 | 黑框眼镜 | 粉色蝴蝶结 | ★★★★★ |
| 物种轮廓 | 金毛犬耳朵下垂 | 布偶猫耳朵竖立 | ★★★★ |
| 体型比例 | 健壮 | 娇小 | ★★★★ |

#### 2.2 辅助标识（高细节画风保留）

| 标识 | 豆包 | 奶盖 |
|------|------|------|
| 毛发质感 | 蓬松长毛 | 重点色 |
| 眼睛颜色 | 深棕色 | 蓝色 |
| 服装细节 | 工装裤口袋 | 针织衫纹理 |
| 表情特征 | 温暖大笑 | 傲娇微笑 |

---

### 三、不同画风中的提示词策略

#### 3.1 水彩风中的角色保持

**关键**：强调色彩和配饰，弱化细节描述。

```
一只拟人化的金毛犬，金色毛发，穿着蓝色卫衣，
戴着黑框眼镜，温暖的微笑，
[动作描述]，[场景描述]，
水彩插画风格，柔和的笔触，温暖的色调
```

**注意**：
- 保留"金毛犬""蓝色卫衣""黑框眼镜"三个核心标识
- 不需要描述"工装裤口袋"等细节（水彩风会模糊）
- 强调色彩关键词

#### 3.2 Q版卡通风中的角色保持

**关键**：强调色彩、配饰和物种轮廓。

```
一只Q版拟人化的金毛犬，大头小身体，金色毛发，
穿着蓝色卫衣，戴着黑框眼镜，萌萌的开心表情，
头身比2:1，[动作描述]，[场景描述]，
Q版卡通风格，线条简洁，色彩明亮
```

**注意**：
- Q版中体型差异会被夸张，利用这一点
- 眼镜和蝴蝶结在Q版中应该更大更明显
- 物种特征（耳朵形状）是重要区分点

#### 3.3 国风水墨风中的角色保持

**关键**：强调轮廓和意境，色彩用墨色变化代替。

```
一只拟人化的金毛犬，穿着蓝色汉服（保留蓝色标识），
戴着黑框眼镜（现代化配饰的反差萌），
水墨画风格，留白写意，墨色浓淡变化
```

**注意**：
- 水墨风中可以用服装变化来保留色彩标识
- 配饰保留但可以简化
- 利用"现代配饰+古典画风"的反差增强识别度

---

### 四、跨画风一致性检查

#### 4.1 检查清单

将同一角色在不同画风中的图片并排，检查：

| 检查项 | 通过标准 |
|--------|---------|
| 主色调一致？ | 一眼能看出"这是金色/米白色的角色" |
| 配饰可见？ | 眼镜/蝴蝶结清晰可辨 |
| 物种可辨？ | 能认出是金毛犬/布偶猫 |
| 两个角色可区分？ | 不会把豆包认成奶盖 |
| 整体气质一致？ | 豆包温暖，奶盖傲娇 |

---

### 五、常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| Q版中两个角色太像 | 物种特征不够突出 | 强调耳朵形状差异和体型差异 |
| 水彩风配饰消失 | 细节在水彩中丢失 | 在提示词中强调配饰，降低参考强度 |
| 不同画风性格不一致 | 性格描述不够统一 | 在所有画风中使用相同的情绪关键词 |
| 画风切换太突兀 | 缺乏过渡 | 使用逐步过渡法 |

## 模块七总结

### 核心知识点回顾

1. **基准图创建**：4角度基准图是角色一致性的基础，必须清晰、高质量
2. **角色档案**：完整的档案包含基础信息、视觉特征、性格设定、动作特征、参考图集、提示词模板
3. **多角度参考图**：8个角度覆盖常用镜头需求，确保任何角度都有视觉参考
4. **性格可视化**：通过表情、动作、光影将抽象性格转化为具体视觉表现
5. **跨画风识别**：核心标识（主色调+配饰+物种轮廓）在任何画风中都必须保留
6. **档案维护**：定期更新，保持与实际生成效果一致

### 速查表

| 需求 | 方法 | 关键点 |
|------|------|--------|
| 创建基准图 | 即梦AI文生图 | 4角度，纯白背景 |
| 建立档案 | 完整文档 | 5大维度 |
| 保持一致性 | 三法合一 | 参考图+描述模板+种子 |
| 性格表达 | 视觉对应 | 表情+动作+光影 |
| 跨画风识别 | 核心标识 | 色彩+配饰+轮廓 |
