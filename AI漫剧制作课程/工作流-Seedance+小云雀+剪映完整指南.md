# 实战工作流：Seedance 2.0 + 小云雀 + 剪映 — 15 秒切片漫剧量产手册

> 这是为 **真实日产模式** 写的操作手册。不讲理论，每一步对应一个动作、一个文件、一个时间预算。
> 适用：你已注册三个工具账号、有可投入预算 / 时间，目标是**每周稳定产出 1-3 集 5-10 分钟的 AI 漫剧**。

---

## 0. 整体管线（1 张图说清）

```
 [DeepSeek/文心]      [即梦AI]         [Seedance 2.0]              [小云雀]              [剪映]
 剧本 + 分镜  →  关键帧静态图  →  15s 视频片段（核心戏）  +  15s 视频片段（轴动文戏）→  剪辑/字幕/配音/调色 → 成片
   30 min          60 min                90 min                       60 min                90 min

                                      [合计：约 5.5 小时 / 集 5-8 分钟]
```

**两条视频线并行的设计原因**：
- **Seedance 2.0** 适合**有明确动作 / 镜头语言 / 角色一致性要求**的镜头（打斗、追车、人物特写）→ **质量优先**
- **小云雀** 适合**对话戏 / 旁白叙事 / 风景空镜 / 转场**的镜头 → **效率优先**（自带剧本→视频 Agent，120 积分/天免费）
- 你不会全部用 Seedance（贵且慢），也不会全部用小云雀（动作戏不行），**两者搭配是最优解**

---

## 1. 选题与剧本（30 分钟）

### 1.1 目标设定

一集 5-8 分钟的漫剧 ≈ **20-32 个 15s 切片** ≈ **20-32 个分镜**。

按 8/2 法则拆：
- **70% 小云雀片段**（对话、旁白、风景、转场）≈ 14-22 段
- **30% Seedance 片段**（关键剧情、动作、特写）≈ 6-10 段

### 1.2 选题（用 DeepSeek）

```
Prompt：
请基于 2026 年 5 月抖音热门 AI 漫剧排行榜，给我推荐 5 个最容易出爆款的题材方向，
每个方向给出：
1. 题材名（5字内）
2. 核心爽点（一句话）
3. 适合的画风（写实 / 漫画 / 国风 / 赛博）
4. 预估单集制作成本（基于即梦+Seedance 2.0+小云雀的三工具组合）
5. 当前赛道竞争度（1-5 星）

输出表格。
```

### 1.3 剧本（用 DeepSeek）

```
Prompt：
为 [题材] 写第 1 集剧本，约 1500 字。要求：
1. 严格按 [开场钩子(15s) → 冲突铺设(60s) → 转折(60s) → 高潮(120s) → 钩子结尾(15s)] 五段结构
2. 每段切分成若干 15 秒切片，给每个切片标 [SD] (Seedance) 或 [XYQ] (小云雀)
3. 每个切片包含：场景 / 角色 / 动作 / 关键台词
4. 主线冲突明确，角色不超过 4 个
5. 钩子结尾必须为下一集留悬念

格式示例：
【切片 01】[XYQ] 出租屋·夜·特写
场景：林晓盯着电脑屏幕，键盘连击声
角色：林晓
动作：他抓头发，叹气，关上电脑
台词：旁白「凌晨三点，林晓被项目折磨到崩溃。」
```

**产出文件**：`scripts/ep01-shotlist.md` — 25-30 个分切片清单

## 2. 角色 + 场景资产准备（60 分钟）

### 2.1 角色基准图（即梦 AI）

每个主要角色生成 **3 张基准图**（正面 / 半侧面 / 全身），以便后续 Seedance 角色档案。

```
即梦 AI Prompt 模板：
[角色姓名]，[年龄]岁[男/女]，
[发型描述]，[眼睛描述]，[面部特征]，[身高体型]，
[服饰：上衣 + 裤子 + 鞋]，
[性格映射的微表情：内向→眼神略躲闪 / 强势→挑眉]，
画面要求：[正面胸像 / 半侧面 3/4 视角 / 全身站姿]，
画风：[漫画风/写实风/国风]，
背景：纯白底，柔光打光，色彩饱和度中等，无水印
```

### 2.2 场景母版图（即梦 AI）

每个核心场景出 **1 张母版**：

```
即梦 AI Prompt 模板：
场景：[出租屋客厅 / 公司办公室 / 公园 / ...]
时段：[早晨 / 中午 / 黄昏 / 夜晚]
光线：[自然光 / 暖灯 / 冷光 / 霓虹]
镜头：[35mm 广角 / 50mm 标准 / 长焦特写]
氛围：[温馨 / 紧张 / 压抑 / 浪漫]
画风：[漫画风/写实/赛博]
分辨率：1920x1080，竖屏请改为 1080x1920
```

### 2.3 在 Seedance 2.0 创角色档案

进 Seedance 2.0 → 角色档案 → 上传刚才的 3 张基准图（按正/侧/全身分类）→ 命名 `林晓_v1` 等。

**关键技巧**：每个角色档案最少 3 张图、最多 8 张。少了不准、多了 Seedance 自己会乱选。

**产出文件**：
- `assets/characters/{name}_{view}.png`（每角色 3 张）
- `assets/scenes/{scene}_master.png`（每场景 1 张）
- Seedance 后台已建好 N 个角色档案

## 3. 视频生成（150 分钟）

### 3.1 [SD] Seedance 2.0 — 处理 6-10 段关键剧情

**适用片段**：
- 角色特写（情绪到位的脸部）
- 动作戏（打斗、奔跑、跳跃）
- 关键转折镜头（举枪对峙、表白瞬间）
- 高难度运镜（环绕、推拉、跟拍）

**操作步骤（每个切片 8-12 分钟）**：

1. Seedance 2.0 → 文生视频 / 图生视频
2. **强烈建议用图生视频**：上传即梦生成的关键帧静态图作为首帧
3. 选「角色档案」绑定刚建好的角色
4. 写视频 Prompt（重点是动作 + 镜头语言 + 时长）：

```
镜头语言：[特写 / 中景 / 远景 / 主观镜头]
运镜方式：[固定 / 推拉 / 平移 / 旋转 / 跟拍]
动作描述：[2-3 个连续动作，每个 3-5 秒]
情绪：[平静 / 焦虑 / 愤怒 / 欣喜]
环境互动：[角色与场景如何互动]
时长：15s
画面比例：9:16 竖屏 / 16:9 横屏（按发布平台定）
```

5. 生成 → 等 3-5 分钟 → **必看**：角色一致性 / 嘴型 / 手指 / 镜头流畅度
6. 不行就调 Prompt 再生成（消耗 1.5-2 倍积分）；3 次还不行换小云雀

**质量验收四看**：
- [ ] 角色脸不变形（最关键）
- [ ] 手指不超过 5 根
- [ ] 镜头无突兀跳变
- [ ] 物理逻辑通顺（不穿模、不漂浮）

**产出文件**：`videos/seedance/ep01_sd_{切片号}.mp4`（6-10 个 15s mp4）

---

### 3.2 [XYQ] 小云雀 — 处理 14-22 段轴动文戏

**适用片段**：
- 对话戏（双人三人对白）
- 旁白叙事（角色独白 + 画面切换）
- 风景空镜（建立场景）
- 过渡转场

**操作步骤**：

1. 把第 1 步剧本里所有 [XYQ] 标记的切片，整理成 1 个 docx 文件，**按场景分块**：

```
# 场景 1：出租屋·夜
旁白：凌晨三点，林晓被项目折磨到崩溃。
（特写：键盘连击 → 抓头发 → 关电脑）

# 场景 2：公司走廊·晨
（全景：林晓走过走廊，同事窃窃私语）
苏瑶（OS）：又被骂了？
```

2. 小云雀 → 短剧 Agent → 上传 docx → 选择「自动分镜 + 智能配音 + 自动配图」
3. 等 5-10 分钟 → 进入 **「故事蓝图」** 编辑器
4. **必做的 3 个修改**：
   - 角色形象：改成与即梦/Seedance 一致的描述（最好直接喂入即梦角色基准图链接）
   - 场景一致性：改成与场景母版一致的描述
   - 镜头时长：每段控制在 12-18 秒
5. 生成视频 → 单段 15s 切片导出

**省积分技巧**：
- 第一遍生成只看分镜不看视频（不消耗大积分）
- 确认分镜没问题再「批量生成视频」
- 每天 120 免费积分≈ 12 段。一集 14-22 段意味着**要分 2 天生成**或**充 6-12 元会员**

**产出文件**：`videos/xyq/ep01_xyq_{切片号}.mp4`（14-22 个 15s mp4）

## 4. 剪映合成（90 分钟）

这是把两条线 30 个 15s 视频拼成一集的关键步骤。

### 4.1 项目设置（5 分钟）

剪映专业版 → 新建项目 → 设置：
- 分辨率：1920×1080（横屏）/ 1080×1920（竖屏抖音）
- 帧率：30fps（与 Seedance 输出一致）
- 项目名：`ep01_main`

### 4.2 素材导入与排序（10 分钟）

1. 导入两个文件夹：`videos/seedance/*` + `videos/xyq/*`
2. **按切片号排序**（命名规范是关键，见下方）
3. 全部拖到主时间轴

**强烈建议命名规范**：
```
ep01_01_xyq.mp4    # 第 1 集第 1 切片，小云雀生成
ep01_02_xyq.mp4
ep01_03_sd.mp4     # 第 3 切片，Seedance 生成
ep01_04_xyq.mp4
...
```

### 4.3 切口处理（30 分钟，最耗时）

**核心问题**：Seedance 和小云雀的画风、配色、节奏不一致。直接拼会很跳。

**解决方案**：
- **统一调色**：选中所有素材 → 调色 → 应用同一个 LUT 或预设
  - 推荐预设：「电影感 - 暖调」或「漫画 - 浓郁」
  - 关键参数：饱和度 +10、对比度 +15、阴影 -10
- **添加转场**：每两个切片之间加 0.3-0.5s 转场
  - 同场景：黑场闪白 / 模糊推出（隐藏切口）
  - 切场景：硬切 + 0.5s 黑场（电影感）
  - 时间跳跃：闪白 + 字幕「3 小时后」
- **首尾对齐**：每个 15s 切片头尾各砍 0.3-0.5s（往往有生成糊帧）

### 4.4 字幕（15 分钟）

1. 选中音轨 → 智能字幕 → AI 识别
2. 字体：站酷快乐体 / 思源黑体 / 阿里巴巴普惠体
3. 大小：60-80px（竖屏）/ 36-48px（横屏）
4. 位置：底部居中，距底 80px
5. 描边：黑色 4px + 白色文字（任何画面都看得清）
6. 关键台词：变色 + 放大（"我的目标是星辰大海" → 红色加粗）

### 4.5 配音 / 配乐 / 音效（20 分钟）

**配音**（如果小云雀已经配好就跳过）：
- 剪映 → 文本朗读 → 选音色（推荐：温柔小哥 / 邻家女孩 / 沉稳男声）
- 多角色：每段对话切换音色
- 情绪标注：「（生气）你给我滚！」

**配乐**：
- 剪映音乐库 → 按情绪搜索（紧张 / 浪漫 / 喜剧 / 悲伤）
- 全程铺底乐：音量 -20dB
- 关键节点加重音乐：高潮处临时换激昂 BGM

**音效**：
- 转场刷音（whoosh）
- 关键动作加音效（开门、玻璃碎、心跳）
- 钩子结尾加紧张音效（dum dum dum）

### 4.6 钩子片头 + 钩子片尾（10 分钟）

**片头 5s**：
- 第 1s：黑底 + 标题大字「《剧名》第 X 集」
- 第 2-4s：精彩画面快剪（每帧 0.3s）
- 第 5s：定格在最有冲击力的画面
- 配「咚~」音效

**片尾 8s**：
- 钩子台词："这一切只是开始……"
- 配紧张 BGM
- 字幕「明日继续 · 关注 @你的账号」

### 4.7 导出

剪映 → 导出 → 设置：
- 分辨率：1080P
- 比特率：高（默认或 12-15Mbps）
- 编码：H.264
- 文件名：`ep01_v1.mp4`

## 5. 质量自检清单

发布前过这一遍。每条不达标必须返工：

### 5.1 画面质量
- [ ] 没有 AI 出片常见的"闪烁"（每两秒画面跳一下）
- [ ] 角色脸全程没变形（特别注意 8s/15s 切口）
- [ ] 手指数量正确（每只手 5 根）
- [ ] 没有穿模 / 漂浮物

### 5.2 剧情节奏
- [ ] 前 3 秒有钩子（不是 logo / 片头，是真情节）
- [ ] 全片不超过 8 分钟（抖音 / 视频号最佳长度）
- [ ] 每 30 秒有一个小高潮 / 转折
- [ ] 结尾留钩子（让人想看下一集）

### 5.3 音画同步
- [ ] 配音与口型大致对得上（不要求一帧一帧）
- [ ] BGM 音量不盖过台词
- [ ] 音效与动作对齐（开门声出现在门开的瞬间）

### 5.4 字幕
- [ ] 字幕没有错别字（特别是 AI 识别的）
- [ ] 关键台词有强调（颜色 / 大小 / 动效）
- [ ] 字幕不挡角色脸 / 关键画面

### 5.5 合规
- [ ] 片头或第 1s 字幕标注「本片由 AI 生成」
- [ ] 剧本无敏感词（涉政 / 色情 / 暴力血腥）
- [ ] 无引用未授权 IP（动漫角色、明星形象）

## 6. 成本与时间预算

### 6.1 单集预算（5-8 分钟漫剧）

| 工具 | 用量 | 成本 |
|------|------|------|
| DeepSeek API | 选题+剧本约 50K tokens | ¥1-2 |
| 即梦 AI（角色+场景） | 30 张图 | 免费额度内（或 ¥5） |
| Seedance 2.0 | 6-10 段 15s 视频 | ¥30-60（按积分计） |
| 小云雀 | 14-22 段 15s 视频 | 免费 120 积分 + ¥6-12 充会员 |
| 剪映 | 后期 | 0（专业版免费） |
| **合计** | — | **¥40-80 / 集** |

对比传统漫剧 5-20 万 / 集，**成本下降 99.6% 以上**。

### 6.2 单集时间预算

| 阶段 | 时长 |
|------|------|
| 选题 + 剧本 | 30 min |
| 资产准备（角色+场景图） | 60 min |
| Seedance 视频生成 + 验收 | 90 min |
| 小云雀视频生成 + 验收 | 60 min |
| 剪映合成 + 字幕 + 配乐 | 90 min |
| **合计** | **5.5 小时 / 集** |

熟练后可压缩到 **3.5 小时 / 集**，意味着**单人日产 2 集**是可行的。

### 6.3 量产排班建议

**单人模式**（你自己干）：
- 周一 / 周三 / 周五 各做 1 集 → 月产 12 集
- 每集发布周期：T 日做、T+1 日发

**双人模式**（你 + 1 个剪辑助理）：
- 你专注剧本 + 资产 + 视频生成（4 小时/集）
- 助理专注剪映合成 + 字幕 + 发布（1.5 小时/集）
- → 月产 25-30 集

## 7. 故障速查表

| 现象 | 原因 | 解决 |
|------|------|------|
| Seedance 角色脸跳 | 角色档案图片少 / 不清晰 | 补到 5-8 张高清图 |
| Seedance 出 5s 不出 15s | 默认时长设置错 | 高级设置 → 视频时长 → 15s |
| 小云雀分镜不准 | 剧本格式不规范 | 严格按「场景+角色+动作+台词」分块 |
| 小云雀视频里中文乱码 | Seedance 不擅长中文字 | **绝不在视频里渲染中文**，全部用剪映加字幕 |
| 剪映合成画风跳 | 两个工具风格差异 | 加 LUT 统一 + 加转场过渡 |
| 配音机械感强 | 用了基础 TTS | 上小云雀情感音色 / 剪映克隆音色 |
| 抖音审核驳回 | 缺 AI 标识 | 第 1s 加「本片 AI 生成」字幕 + 视频简介注明 |
| 钩子开头无吸引力 | 把背景介绍放前面 | 把高潮 1s 放最前，再倒叙 |

## 8. 进阶玩法

### 8.1 角色克隆音色（剪映）
- 录 30s 自己/演员的声音 → 剪映 → AI 克隆音色
- 之后所有角色配音都可以用这个音色变 → 比纯 TTS 自然

### 8.2 抖音爆款公式套用
- 标题：「【AI 漫剧】+ 题材 + 钩子句 + 情绪词」
  - 例：「【AI 漫剧】打工人逆袭：被裁员当天，我成了天使投资人最爱的人」
- 封面：高潮画面截图 + 大字标题 + 红黄对比色
- Hashtag：#AI漫剧 #短剧 + 题材 hashtag

### 8.3 系列化（关键！）
- 单集播放量 < 10 万 → 不要硬刚，第 3 集果断弃坑换题材
- 单集播放量 > 50 万 → 立刻拉 8-12 集长篇剧本，每天发 1 集
- 系列连续 5 集 > 50 万 → 进入"持续更新"模式，每周 3 集稳定输出

## 10. 学习路径建议（针对本工作流）

如果你的目标就是「即梦 + Seedance + 小云雀 + 剪映 出漫剧」，**6 周速成路径**：
