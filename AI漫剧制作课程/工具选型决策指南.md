## 二、视频工具选型决策树

```
你的镜头需要什么？
│
├── 需要音画同步（对话场景）
│   └── → Vidu Q3
│       优势：口型对齐、多角色对话
│       劣势：动作场景一般
│
├── 需要物理仿真（打斗/动作）
│   └── → 可灵AI / 海螺AI
│       可灵AI优势：物理效果好、1080P
│       海螺AI优势：高难度动作
│
├── 需要角色一致性（跨镜头）
│   └── → Seedance 2.0
│       优势：角色档案驱动、Draft模式
│       劣势：需要付费
│
├── 需要快速预览
│   └── → Seedance 2.0 Draft模式
│       优势：480P快速预览，节省60%成本
│
└── 不确定用什么
    └── → 可灵AI（通用性最强）
        覆盖大部分场景，质量稳定
```

## 三、图像工具选型决策树

```
你的需求是什么？
│
├── 需要角色一致性
│   ├── 有参考图 → 即梦AI（角色参考功能）
│   └── 无参考图 → 即梦AI + LoRA训练
│
├── 需要批量生成
│   └── → 即梦AI（批量生成功能最强）
│
├── 需要精细控制
│   └── → ComfyUI + ControlNet
│       优势：完全可控
│       劣势：学习曲线陡
│
├── 需要免费额度
│   └── → 通义万相（免费额度最多）
│
└── 需要特定风格
    ├── 国风/水墨 → 即梦AI / 通义万相
    ├── 日系动漫 → Kolors
    └── 写实风格 → 即梦AI / 混元
```
