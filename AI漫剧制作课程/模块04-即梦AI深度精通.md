### 一、即梦AI是什么

即梦AI（英文名Dreamina）是字节跳动于2024年推出的AI创作平台，网址：**jimeng.jianying.com**（也有人通过抖音App内的"即梦"入口访问）。

**它能做什么？**

| 能力 | 说明 |
|------|------|
| 文生图 | 输入文字描述，AI生成图片 |
| 图生图 | 上传一张图，AI在此基础上变化风格或修改内容 |
| 文生视频 | 输入文字描述，AI生成短视频片段（通常4-8秒） |
| 图生视频 | 上传一张静态图，AI让它"动起来" |
| 智能编辑 | 局部重绘、扩图、去背景等后期操作 |
| 角色参考 | 锁定角色外观，在不同画面中保持一致性 |

**为什么漫剧制作选即梦？**

- **免费额度充足**：新用户每天有大量免费生成次数
- **中文理解强**：对中文提示词的理解优于Midjourney等海外工具
- **风格多样**：内置多种动漫风格预设，开箱即用
- **生态整合**：与剪映（CapCut）无缝衔接，生成内容可直接导入剪辑
- **角色一致性**：提供"角色参考"功能，解决AI漫剧最大痛点——同一角色跨镜头形象不统一

### 二、完整链路闭环：从灵感到成片

一个完整的AI漫剧制作流程如下：

```
[创意构思] → [剧本撰写] → [分镜设计] → [关键帧生成] → [视频合成] → [配音配乐] → [剪辑成片] → [发布分发]
    ↑                                              ↑                              ↑
    |              即梦AI主要负责这一段              |         剪映负责这一段        |
```

**详细拆解**：

**Step 1：创意构思（5分钟）**
- 确定漫剧题材（如：都市逆袭、校园恋爱、古风仙侠）
- 确定目标受众与平台（抖音短剧、B站番剧、微信视频号）
- 确定视觉风格（日漫风、国漫风、赛博朋克风等）

**Step 2：剧本撰写（30分钟）**
- 用AI辅助（如ChatGPT/Kimi）生成分集剧本
- 每集控制在1-3分钟（60-120个分镜画面）
- 标注每个镜头的角色、场景、动作、情绪

**Step 3：分镜设计（20分钟）**
- 将剧本转化为分镜表
- 每个镜头写明：画面描述、镜头类型（特写/中景/远景）、角色列表
- 这一步直接决定了后续AI生成的提示词质量

**Step 4：关键帧生成（核心步骤，即梦AI）**
- 打开即梦AI，进入"文生图"模块
- 逐镜头输入提示词，生成关键帧画面
- 使用"角色参考"功能保持角色一致性
- 使用"图生图"微调不满意的画面

**Step 5：视频合成（即梦AI）**
- 将关键帧画面上传到"图生视频"模块
- 为每个镜头添加运镜描述（如"缓慢推进""镜头左移"）
- 生成4-8秒的动态视频片段

**Step 6：配音配乐（剪映/外部工具）**
- 使用剪映的AI配音功能生成旁白和对白
- 从剪映素材库选择背景音乐
- 添加音效（脚步声、风声、打斗声等）

**Step 7：剪辑成片（剪映）**
- 将所有视频片段按分镜顺序排列
- 添加转场效果、字幕、特效
- 调色统一画面风格
- 导出最终视频

### 常见问题

**Q1：即梦AI需要付费吗？**
A：基础功能免费使用，每天有免费额度（通常每天可免费生成数十张图片和若干视频）。高级功能（如更高分辨率、更快生成速度）需要开通会员。

**Q2：生成的图片有版权问题吗？**
A：即梦AI生成的图片版权归用户所有，可以用于商业用途。但建议不要直接使用明显的真人肖像或品牌标志。

**Q3：网页版和App版有什么区别？**
A：功能基本一致，但网页版操作更方便（屏幕大，适合批量操作），App版适合随时随地快速生成。

**Q4：生成速度慢怎么办？**
A：高峰期（晚上8-11点）生成速度会变慢，建议错峰使用或开通会员享受优先队列。

## 第26课：即梦AI文生图基础——提示词编写规范与参数调节

### 学习目标

1. 掌握即梦AI提示词的编写规范与结构化模板
2. 理解各参数（分辨率、风格、采样步数等）的含义与调节方法
3. 能独立编写高质量提示词并生成符合预期的画面
4. 了解负面提示词的使用方法

### 一、提示词编写规范

#### 1.1 提示词的基本结构

一个高质量的提示词应包含以下要素（按优先级排列）：

```
[主体描述] + [场景/环境] + [风格/画风] + [镜头/构图] + [光照/氛围] + [质量修饰词]
```

**示例拆解**：

```
一个银白色长发的少女（主体），站在樱花盛开的日式庭院中（场景），日系动漫画风（风格），
中景正面镜头（构图），柔和的逆光照射（光照），超高质量，精致细节（质量修饰）
```

#### 1.2 提示词编写七步法

**第一步：确定主体**
- 明确画面中的核心人物或物体
- 描述外貌特征：发型、发色、眼睛颜色、服装、表情、年龄
- 示例：`一个20岁左右的黑发青年男子，穿着深蓝色西装，表情严肃`

**第二步：设定场景**
- 描述主体所处的环境
- 包括：室内/室外、具体地点、背景元素
- 示例：`站在现代都市的摩天大楼楼顶，身后是夕阳下的城市天际线`

**第三步：指定风格**
- 明确画风：动漫、写实、油画、水彩、像素等
- 可以引用具体风格参考：宫崎骏风格、新海诚风格、赛博朋克风格
- 示例：`日系精致动漫风格，类似新海诚电影的画面质感`

**第四步：定义构图**
- 镜头类型：特写（脸部）、中景（半身）、全景（全身）、远景（大场景）
- 角度：正面、侧面、俯视、仰视
- 示例：`中景侧面45度角，角色位于画面左侧三分之一处`

**第五步：设置光照**
- 光源方向：顺光、逆光、侧光
- 光线类型：自然光、人工光、霓虹灯光
- 氛围：温暖、冷峻、梦幻、阴暗
- 示例：`金色夕阳从右侧照射，形成温暖的侧光效果`

**第六步：添加细节**
- 特殊元素：飘动的头发、飞舞的花瓣、水面倒影
- 材质质感：丝绸光泽、金属反光、玻璃透明感
- 示例：`微风吹起发丝和衣角，空气中飘浮着樱花花瓣`

**第七步：质量修饰**
- 通用质量词：高质量、超高清、8K、精致细节、电影感
- 避免的缺陷词：在负面提示词中注明
- 示例：`超高质量，8K分辨率，精致的人物细节，电影级画面`

#### 1.3 完整提示词模板

**通用模板**：
```
[角色描述], [动作/姿态], [场景环境], [画风风格], [镜头构图], [光照氛围], [细节补充], [质量修饰]
```

**漫剧专用模板**：
```
[角色名+外貌], [情绪/表情], [正在做什么], [在什么地方], [什么时间], [动漫风格], [镜头类型], [画面质量]
```

**实例——漫剧第一集第一幕**：
```
林晓（25岁黑色短发青年，穿灰色卫衣），疲惫的表情，坐在电脑前敲代码，
在狭小的出租屋里，深夜，日系都市动漫风格，中景正面，台灯暖光照亮面部，
窗外城市灯火模糊可见，超高质量，精致细节
```

#### 1.4 负面提示词（反向提示词）

负面提示词告诉AI"不要生成什么"，用于避免常见缺陷：

**常用负面提示词**：
```
低质量，模糊，变形，多余手指，畸形手，扭曲的脸，水印，文字，标志，
过度饱和，噪点，伪影，不自然的姿势，解剖学错误
```

**在即梦AI中的使用方法**：
- 文生图界面中，找到"反向提示词"或"负面提示词"输入框
- 将不需要的元素填入该框
- 不是所有模型版本都支持负面提示词，如果不显示该选项则跳过

### 二、参数调节详解

#### 2.1 图片比例（宽高比）

| 比例 | 适用场景 | 推荐用途 |
|------|---------|---------|
| 1:1 | 方形构图，头像，社交媒体封面 | 角色立绘、封面图 |
| 4:3 | 传统横屏，经典电影比例 | 横屏漫剧、场景图 |
| 16:9 | 宽屏横屏，电影级构图 | 横屏视频关键帧 |
| 9:16 | 竖屏，手机全屏 | 抖音/快手竖屏漫剧 |
| 3:4 | 竖屏但不那么极端 | 漫画分格、竖屏海报 |

**漫剧制作推荐**：
- 抖音/快手竖屏短剧 → **9:16**
- B站横屏番剧 → **16:9**
- 微信视频号 → **9:16** 或 **3:4**

#### 2.2 模型版本选择

即梦AI会不断更新模型版本，常见版本：

| 模型 | 特点 | 适用场景 |
|------|------|---------|
| 通用v1.0 | 基础模型，速度快 | 快速草图、测试提示词 |
| 通用v2.0 | 质量提升，细节更好 | 正式生成、漫剧关键帧 |
| 动漫专用模型 | 专门针对动漫风格优化 | 日漫/国漫风格漫剧 |
| 写实模型 | 照片级真实感 | 写实风格短剧 |

**选择建议**：做漫剧优先使用**动漫专用模型**或**通用v2.0**。

#### 2.3 生成数量

- 可选择每次生成1/2/4张
- **建议选择4张**：同一提示词生成4张，从中挑选最佳效果
- 这是"抽卡"策略——AI生成有随机性，多生成几张能大幅提高满意度

#### 2.4 采样步数（Steps）

采样步数决定AI"思考"的深度：
- **低步数（15-20步）**：速度快，但细节可能不够丰富
- **中步数（25-30步）**：平衡速度与质量，**推荐日常使用**
- **高步数（35-50步）**：细节最丰富，但速度慢

**漫剧制作建议**：默认使用25-30步，最终精修时可提高到40步。

> 📸 **[截图位置说明：26-03]** 插入参数设置面板截图，标注模型选择、生成数量、采样步数等参数位置。

#### 2.5 引导强度（CFG Scale）

引导强度控制AI对提示词的"忠实度"：
- **低值（3-5）**：AI更自由发挥，可能偏离提示词但更有创意
- **中值（7-8）**：平衡创意与忠实，**推荐使用**
- **高值（10-15）**：严格遵循提示词，但可能显得生硬

#### 2.6 随机种子（Seed）

- 每次生成都有一个随机种子值
- 固定种子可以在相同提示词下获得相似结果
- **使用场景**：找到一张满意的图后，记录其种子值，微调提示词时固定种子可以保持整体构图不变

### 三、提示词优化实战

#### 3.1 从模糊到精确的优化过程

**初始（模糊）提示词**：
```
一个女孩在花园里
```
问题：太模糊，AI会随意发挥，结果不可控。

**第一次优化（加入角色细节）**：
```
一个16岁的粉色双马尾少女，穿着白色连衣裙，站在花园里
```

**第二次优化（加入场景细节）**：
```
一个16岁的粉色双马尾少女，穿着白色连衣裙，站在盛开着红玫瑰的英式花园中，
阳光明媚的午后
```

**第三次优化（加入风格和构图）**：
```
一个16岁的粉色双马尾少女，穿着白色连衣裙，站在盛开着红玫瑰的英式花园中，
阳光明媚的午后，日系动漫画风，中景正面镜头，柔和的自然光照，超高质量
```

**第四次优化（加入情绪和细节）**：

**Step 2：分镜表转提示词**
- 将分镜表中的每一行转化为提示词
- 保持风格词和质量词不变，只替换角色动作和场景

**Step 3：批量生成**

- 检查角色外貌是否一致
- 检查场景是否连贯
- 不满意的重新生成或微调提示词

### 常见问题

**Q1：生成的手总是变形怎么办？**
A：在负面提示词中加入"畸形手，多余手指，变形的手"；尽量避免手部特写镜头；或者用手部参考图辅助生成。

**Q2：提示词太长AI会忽略后面的内容吗？**
A：会。提示词越靠前的词权重越高。把最重要的描述放在前面，次要的放后面。

**Q3：为什么同样的提示词每次生成结果不同？**
A：因为AI使用随机噪声作为起点，每次的随机种子不同。如果想复现某张图的效果，需要固定随机种子值。

**Q4：中文和英文提示词哪个效果好？**
A：即梦AI对中文的理解已经很好，日常使用中文即可。但某些特定风格词用英文可能更精确（如"cyberpunk""photorealistic"），可以中英混合使用。

## 第27课：即梦AI图生图进阶——风格迁移与画面精修

### 学习目标

1. 掌握图生图功能的核心原理与使用方法
2. 学会使用风格迁移将照片/草图转化为动漫风格
3. 掌握画面精修技巧，对不满意的生成结果进行微调
4. 理解"重绘强度"参数的调节策略

### 一、图生图的核心原理

**什么是图生图？**

图生图（Image-to-Image）是在一张已有图片的基础上，通过AI重新绘制来生成新图片的技术。你可以把它理解为：你给AI一张"参考底稿"，AI在这个基础上进行"重新创作"。

**核心参数——重绘强度（Denoising Strength）**

重绘强度是图生图最关键的参数，控制AI对原图的"改变程度"：

| 重绘强度 | 效果 | 适用场景 |
|---------|------|---------|
| 0.1-0.3 | 轻微修改，保留原图大部分内容 | 微调色调、修复小瑕疵 |
| 0.4-0.6 | 中等修改，保留构图但改变细节 | 风格迁移、换装、换背景 |
| 0.7-0.8 | 较大修改，保留大致轮廓 | 大幅风格转换 |
| 0.9-1.0 | 几乎完全重绘，仅保留构图暗示 | 从草图生成精细画面 |

### 二、风格迁移操作详解

#### 2.1 照片转动漫风格

**场景**：你有一张真人照片（或网络素材），想把它转化为动漫风格用于漫剧。

**操作步骤**：

**Step 1：进入图生图模块**
- 在即梦AI首页，点击"图生图"卡片
- 进入图生图工作界面

**Step 2：上传参考图片**
- 点击"上传图片"按钮或直接将图片拖入上传区域
- 等待图片上传完成
- 图片会显示在左侧预览区

> 📸 **[截图位置说明：27-01]** 插入图生图界面截图，标注上传区域和图片预览位置。

**Step 3：设置重绘强度**
- 找到"重绘强度"或"变化程度"滑块
- **照片转动漫建议设置：0.6-0.7**
  - 太低（0.3以下）：还是照片质感，不够动漫
  - 太高（0.8以上）：可能丢失人物特征
  - 0.6-0.7是照片转动漫的甜蜜点

**Step 4：输入风格提示词**
在提示词框中输入：
```
日系精致动漫风格，高质量动漫插画，清晰的线条，鲜艳的色彩，动漫人物渲染
```

**Step 5：选择模型和参数**
- 模型：选择动漫专用模型（如有）或通用v2.0
- 图片比例：保持与原图一致
- 生成数量：4张

**Step 6：生成并选择**
- 点击"生成"按钮
- 等待生成完成
- 从4张结果中选择最佳效果

> 📸 **[截图位置说明：27-02]** 插入照片转动漫的前后对比效果图，左边原照片，右边动漫化结果。

#### 2.2 草图/线稿上色

**场景**：你有手绘草图或AI生成的线稿，想让它变成完整的彩色画面。

**操作步骤**：

**Step 1：上传线稿**
- 将线稿图片上传到图生图模块

**Step 2：设置重绘强度**
- **线稿上色建议设置：0.7-0.85**
- 因为线稿信息较少，需要AI更多地"填充"颜色和细节

**Step 3：输入详细提示词**
```
精致的动漫上色，鲜艳饱满的色彩，柔和的光影效果，干净的线条，
[角色描述]，[场景描述]，高质量动漫插画
```

**Step 4：生成并迭代**
- 生成4张，选择上色效果最好的
- 如果颜色不对，调整提示词中颜色相关的描述
- 如果构图偏了，降低重绘强度重试

#### 2.3 旧漫剧画面风格统一

**场景**：你之前的漫剧画面风格不统一（有的偏日漫，有的偏国漫），需要统一风格。

**操作步骤**：

**Step 1：确定目标风格**
- 选择一个你最满意的画面作为"风格基准"

**Step 2：逐张风格迁移**
- 将需要调整的图片上传到图生图
- 设置重绘强度为0.4-0.5（保留内容，只改风格）
- 输入统一的风格描述词

**Step 3：建立风格描述模板**
为确保一致性，所有图片使用完全相同的风格描述：
```
统一风格描述示例：
日系精致动漫风格，柔和的色彩渐变，清晰的轮廓线条，统一的色调为暖色系，
类似京都动画的画面质感，高质量，8K
```

### 三、画面精修技巧

#### 3.1 微调表情和动作

**场景**：生成的角色表情不对（应该是微笑但生成了严肃脸）。

**方法**：使用图生图 + 低重绘强度

**操作步骤**：
1. 将需要修改的图片上传到图生图
2. 设置重绘强度为 **0.3-0.4**（轻微修改）
3. 在提示词中强调需要修改的部分：
   ```
   [原图描述]，角色面带温柔的微笑，眼神柔和，愉悦的表情
   ```
4. 生成后对比，如果变化太小就提高到0.5，变化太大就降低到0.25

#### 3.2 更换背景

**场景**：角色很好但背景不满意，想换一个场景。

**方法**：使用图生图 + 中重绘强度 + 背景描述

**操作步骤**：
1. 上传原图
2. 设置重绘强度为 **0.5-0.6**
3. 在提示词中详细描述新背景：
   ```
   [角色描述保持不变]，背景是夜晚的东京涩谷十字路口，
   霓虹灯光牌匾，人流涌动，繁华都市夜景
   ```
4. 生成后检查角色是否保持不变，如果角色也变了，降低重绘强度

#### 3.3 增强画面细节

**场景**：图片整体不错但细节不够（衣服纹理模糊、背景元素太少）。

**方法**：使用图生图 + 低重绘强度 + 细节增强词

**操作步骤**：
1. 上传原图
2. 设置重绘强度为 **0.25-0.35**
3. 在原提示词基础上增加细节词：
   ```
   [原提示词]，增强细节，精细的服装纹理，丰富的背景元素，
   高质量渲染，精致的画面细节
   ```

#### 3.4 修复AI常见缺陷

| 缺陷 | 修复方法 |
|------|---------|
| 手指变形 | 上传原图，重绘强度0.3-0.4，提示词强调"正常的手部比例，五根手指" |
| 眼睛不对称 | 上传原图，重绘强度0.3，提示词强调"对称的眼睛，精致的眼部细节" |
| 多余肢体 | 上传原图，重绘强度0.4-0.5，负面提示词加入"多余肢体，畸形" |
| 脸部崩坏 | 如果崩坏严重，建议重新生成而非修复 |
| 背景混乱 | 使用更换背景方法，重绘强度0.5-0.6 |

> 📸 **[截图位置说明：27-03]** 插入修复前后对比图，展示手指修复、表情修正等效果。

### 四、高级图生图技巧

#### 4.1 多次迭代精修

对于重要画面（如封面、关键剧情镜头），可以进行多轮迭代：

```
第一轮：文生图生成基础画面（选择最接近预期的）
    ↓
第二轮：图生图调整构图和角色姿态（重绘强度0.5）
    ↓
第三轮：图生图微调表情和细节（重绘强度0.3）
    ↓
第四轮：图生图增强画质和细节（重绘强度0.2）
```

每轮只解决一到两个问题，不要贪多。

#### 4.2 种子值+图生图组合

1. 先用文生图找到满意的种子值
2. 用该种子值生成基础画面
3. 将基础画面上传到图生图，固定种子值
4. 微调提示词，重绘强度设低（0.2-0.3）
5. 这样可以在保持整体构图的前提下进行微调

### 常见问题

**Q1：图生图和文生图应该先用哪个？**
A：建议先用文生图找到接近预期的基础画面，再用图生图进行精修。不要试图用图生图从一张完全不相关的图片"变"出想要的画面。

**Q2：重绘强度设多少最合适？**
A：没有万能值。照片转动漫用0.6-0.7，微调用0.2-0.3，换背景用0.5-0.6。多试几次就能找到感觉。

**Q3：上传的图片分辨率有要求吗？**
A：建议上传至少1024×1024的图片。太小的图片会影响生成质量。如果原图太小，可以先用AI放大工具（如Upscayl）放大后再上传。

**Q4：图生图会改变原图的版权吗？**
A：图生图生成的是全新图片，与原图在像素层面完全不同。但如果是基于他人作品进行风格迁移，建议注意版权问题。

### 课后练习

1. **风格迁移练习**：找一张自己的照片（或网络素材），分别转化为日漫风格、国漫水墨风格、赛博朋克风格
2. **精修练习**：用文生图生成一张图，然后用图生图进行至少3轮迭代精修
3. **换背景练习**：将同一角色放在3个不同场景中，保持角色不变
4. **缺陷修复练习**：故意生成一张有缺陷的图片（如手指变形），然后用图生图修复

## 第28课：即梦AI角色一致性技术——跨镜头形象稳定方案

### 学习目标

1. 理解AI漫剧中角色一致性的核心挑战
2. 掌握即梦AI"角色参考"功能的完整使用步骤
3. 学会多方案组合确保角色跨镜头形象稳定
4. 能独立为一个漫剧角色建立一致性工作流

### 一、为什么角色一致性是AI漫剧的最大挑战

**问题本质**：AI每次生成图片都是"从零开始"的随机过程，即使使用完全相同的提示词，每次生成的角色外貌都会有差异——发型可能略有不同、脸型可能变化、服装细节可能不一致。

**在漫剧中这意味着什么？**
- 第1集第1幕的主角是圆脸、大眼睛
- 第1集第5幕的同一主角变成了长脸、小眼睛
- 观众会感觉"这不是同一个人"，严重影响观看体验

**解决方案概览**：

| 方案 | 原理 | 效果 | 操作难度 |
|------|------|------|---------|
| 角色参考功能 | 上传角色参考图，AI锁定外貌特征 | ★★★★★ | 简单 |
| 固定提示词模板 | 用完全一致的文字描述角色 | ★★★☆☆ | 简单 |
| 种子值固定 | 固定随机种子保持相似性 | ★★☆☆☆ | 中等 |
| 图生图迭代 | 基于已有角色图进行微调 | ★★★★☆ | 中等 |
| 多方案组合 | 以上方法综合使用 | ★★★★★ | 较复杂 |

### 二、角色参考功能使用步骤（核心方法）

**这是目前最有效的角色一致性方案，务必掌握。**

#### 2.1 创建角色基准图

**Step 1：设计角色初始形象**
- 使用文生图功能，输入详细的角色描述
- 生成4-8张图片，选择最符合预期的一张作为"角色基准图"
- **选择标准**：面部清晰、姿态端正、光线均匀、背景简洁

**基准图提示词模板**：
```
[角色名]，[年龄]岁，[性别]，[发型+发色]，[眼睛颜色]，[面部特征]，
穿着[服装描述]，[表情]，[姿势]，白色/简洁背景，日系动漫风格，
正面或3/4侧面，清晰的面部特征，高质量角色立绘
```

**示例**：
```
林晓，25岁，男性，黑色短碎发，深棕色眼睛，棱角分明的面部轮廓，
穿着深灰色连帽卫衣和黑色牛仔裤，平静的表情，站立姿势，
白色简洁背景，日系都市动漫风格，正面3/4侧面，清晰的面部特征，
高质量角色立绘，全身像
```

> 📸 **[截图位置说明：28-01]** 插入角色基准图示例，展示一张高质量的角色立绘。

**Step 2：选择最佳基准图**
从生成的多张图片中选择最佳基准图的标准：
- ✅ 面部特征清晰可辨
- ✅ 发型、发色明确
- ✅ 服装细节完整
- ✅ 光线均匀，没有强烈阴影
- ✅ 姿态自然，不扭曲
- ❌ 避免选择有手部遮挡面部的图
- ❌ 避免选择背景过于复杂的图
- ❌ 避免选择角度过于极端的图（纯正面或纯背面）

#### 2.2 使用角色参考功能

**Step 3：进入文生图界面**
- 在即梦AI首页点击"文生图"

**Step 4：开启角色参考**
- 在文生图界面左侧（或提示词输入框上方），找到**"角色参考"**开关
- 点击开启，会出现上传参考图的区域

> 📸 **[截图位置说明：28-02]** 插入文生图界面中"角色参考"开关的位置截图，用红框标注。

**Step 5：上传角色基准图**
- 点击"上传参考图"按钮
- 选择你在Step 1中创建的角色基准图
- 等待上传完成
- 上传后可以在预览区看到参考图

> 📸 **[截图位置说明：28-03]** 插入角色参考图上传后的界面截图，标注参考图预览位置。

**Step 6：设置参考强度**
- 上传参考图后，会出现一个"参考强度"或"相似度"滑块
- **建议设置：0.7-0.85**
  - 0.5以下：参考效果弱，角色可能不够像
  - 0.7-0.85：保持角色特征的同时允许姿态变化（推荐）
  - 0.9以上：角色非常像，但姿态可能过于僵硬

**Step 7：输入新画面的提示词**
- 在保持角色外貌描述词不变的基础上，修改动作、场景、情绪等
- **关键**：提示词中的角色描述要与基准图一致

**示例——同一角色不同场景**：

基准图描述：
```
林晓，25岁，黑色短发青年，深灰色卫衣
```

新画面提示词：
```
林晓，25岁，黑色短发青年，穿着深灰色卫衣，焦虑的表情，
坐在电脑前敲代码，在昏暗的出租屋里，深夜，屏幕光照亮面部，
日系都市动漫风格，中景正面，高质量
```

**Step 8：生成并验证**
- 点击"生成"
- 检查生成结果中角色是否与基准图一致
- 如果不够像，提高参考强度重试

#### 2.3 批量生成同一角色多镜头

**工作流**：

```
角色基准图（1张）
    │
    ├── 镜头1：正面微笑表情 → 生成4张，选1张
    ├── 镜头2：侧面严肃表情 → 生成4张，选1张
    ├── 镜头3：背面行走姿态 → 生成4张，选1张
    ├── 镜头4：特写眼部 → 生成4张，选1张
    └── 镜头5：全景站立 → 生成4张，选1张
```

每个镜头都使用相同的基准图作为角色参考，确保外貌一致。

### 三、多镜头形象稳定进阶方案

#### 3.1 提示词锚定法

**核心思想**：为角色创建一个固定的"外貌锚定描述"，在所有镜头中保持完全一致。

**建立角色锚定描述**：
```
【林晓-外貌锚定】
面部：25岁男性，棱角分明的脸型，深棕色瞳孔，浓眉
发型：黑色短碎发，略微向右偏分，鬓角修剪整齐
体型：中等身材，身高约178cm
标志性特征：左眉上方有一道浅浅的疤痕
```

**使用规则**：
- 每个镜头的提示词都必须包含以上锚定描述
- 只修改动作、场景、情绪等变量部分
- 锚定描述永远放在提示词最前面（权重最高）

#### 3.2 图生图接力法

**核心思想**：用上一个镜头的生成结果作为下一个镜头的参考底图。

**操作流程**：
```
镜头1（文生图+角色参考）→ 满意的图片A
    │
    ↓
镜头2（图生图，上传图片A，重绘强度0.4-0.5）→ 满意的图片B
    │
    ↓
镜头3（图生图，上传图片B，重绘强度0.4-0.5）→ 满意的图片C
    ...
```

**优点**：角色一致性极高，因为每一步都基于上一步的结果。
**缺点**：如果中间某一步出了问题，后续都会受影响；且风格可能逐渐偏移。

**最佳实践**：每隔3-5个镜头，重新使用角色基准图"校准"一次。

#### 3.3 多角度参考图法

**核心思想**：为同一角色创建多个角度的参考图（正面、侧面、背面），在不同镜头中使用对应角度的参考。

**操作步骤**：

**Step 1：创建多角度基准图**
- 用文生图分别生成角色的正面、3/4侧面、纯侧面、背面图
- 提示词模板：
  ```
  [角色锚定描述]，正面全身立绘，白色背景，角色设计图风格
  [角色锚定描述]，3/4侧面全身立绘，白色背景，角色设计图风格
  [角色锚定描述]，纯侧面全身立绘，白色背景，角色设计图风格
  [角色锚定描述]，背面全身立绘，白色背景，角色设计图风格
  ```

**Step 2：根据镜头角度选择参考图**
- 正面镜头 → 使用正面参考图
- 侧面镜头 → 使用侧面参考图
- 背面镜头 → 使用背面参考图

> 📸 **[截图位置说明：28-05]** 插入同一角色四个角度的参考图示例。

#### 3.4 服装锁定技巧

**问题**：AI有时会改变角色的服装（明明穿的是卫衣，生成结果变成了西装）。

**解决方案**：
1. 在提示词中**极其详细地描述服装**：
   ```
   穿着深灰色连帽卫衣，卫衣正面有一个小口袋，拉链半开，
   内搭白色T恤，下穿黑色修身牛仔裤，脚穿白色运动鞋
   ```
2. 在角色参考功能中使用包含该服装的参考图
3. 如果服装还是变了，用图生图+低重绘强度（0.3）修复

### 四、多角色场景的一致性

**当一个画面中出现多个角色时，如何保持每个角色的一致性？**

#### 4.1 逐角色叠加法

**操作步骤**：
1. 先用角色A的参考图生成只有角色A的场景
2. 再用角色B的参考图+图生图，将角色B添加到画面中
3. 重绘强度设置为0.4-0.5，确保添加角色B的同时不破坏角色A

#### 4.2 多角色提示词分离法

在提示词中清晰区分不同角色：
```
画面左侧：林晓（25岁黑色短发青年，穿深灰色卫衣，严肃表情）
画面右侧：苏瑶（22岁银白色长发少女，穿白色连衣裙，微笑表情）
两人面对面站立在樱花树下，日系动漫风格，中景，高质量
```

**注意**：多角色场景的生成难度较高，可能需要多次尝试。

### 常见问题

**Q1：角色参考功能对所有模型都有效吗？**
A：大部分模型版本支持，但效果可能有差异。建议使用最新版本的模型获得最佳效果。

**Q2：参考图可以是真人照片吗？**
A：可以，但效果不如动漫风格的参考图。建议先用图生图将照片转为动漫风格，再作为参考图使用。

**Q3：角色参考强度设为最高（1.0）效果最好吗？**
A：不一定。过高的参考强度可能导致角色姿态过于僵硬，缺乏变化。0.7-0.85通常是最优范围。

**Q4：如何管理大量角色的参考图？**
A：建议建立一个"角色资产文件夹"，每个角色一个子文件夹，包含：多角度基准图、外貌锚定描述文档、各镜头的最终选择图。

3. **一致性对比**：分别用"仅提示词""提示词+角色参考""图生图接力"三种方法生成同一角色的3张图，对比一致性效果

## 第29课：即梦AI场景资产库搭建——核心场景"母版图"批量生成

### 学习目标

1. 理解场景资产库对漫剧制作效率的重要性
2. 掌握核心场景"母版图"的设计与生成方法
3. 学会基于母版图批量衍生同一场景的不同角度/时间/天气版本
4. 能独立搭建一个漫剧的基础场景资产库

### 一、为什么要建立场景资产库

**问题**：漫剧中同一个场景会反复出现（如主角的家、学校教室、城市街道）。如果每次需要该场景时都从零生成，不仅效率低，而且场景细节会不一致（上一幕窗外是白天，下一幕变成了晚上；上一幕桌上有杯子，下一幕杯子消失了）。

**解决方案**：建立"场景资产库"——为每个核心场景创建一张高质量的"母版图"，然后基于母版图衍生出各种变体。

**场景资产库的结构**：
```
场景资产库/
├── 场景01-主角出租屋/
│   ├── 母版图-全景.png
│   ├── 变体-夜晚.png
│   ├── 变体-特写桌面.png
│   ├── 变体-窗口视角.png
│   └── 场景描述.txt
├── 场景02-公司办公室/
│   ├── 母版图-全景.png
│   ├── 变体-会议室.png
│   └── 场景描述.txt
├── 场景03-城市街道/
│   ├── 母版图-白天.png
│   ├── 变体-夜晚霓虹.png
│   └── 场景描述.txt
└── ...
```

### 二、核心场景"母版图"生成方法

#### 2.1 确定漫剧的核心场景

首先列出你的漫剧需要的所有场景，按出现频率排序：

**示例——都市逆袭题材漫剧**：

| 优先级 | 场景名称 | 出现频率 | 说明 |
|-------|---------|---------|------|
| ★★★★★ | 主角出租屋 | 每集都有 | 日常场景，需要多个角度 |
| ★★★★★ | 城市街道 | 每集都有 | 转场、行走场景 |
| ★★★★☆ | 公司办公室 | 大部分集 | 工作场景 |
| ★★★☆☆ | 咖啡厅 | 偶尔出现 | 对话场景 |
| ★★★☆☆ | 医院 | 剧情需要 | 特定剧情场景 |
| ★★☆☆☆ | 豪宅 | 后期剧情 | 对比场景 |

**建议**：先为前3-5个高频场景创建母版图，后续按需补充。

#### 2.2 母版图的设计原则

**母版图是什么？**
母版图是该场景的"标准照"——一个最能代表该场景特征的全景画面，后续所有变体都基于它衍生。

**母版图的设计要求**：

1. **全景视角**：能展示场景的全貌
2. **中性光线**：不要太暗或太亮，方便后续调整
3. **无角色**：纯场景，不包含人物（这样后续可以自由添加角色）
4. **细节丰富**：包含场景的关键元素和装饰细节
5. **风格统一**：与你的漫剧整体画风一致

#### 2.3 母版图生成实操

**以"主角出租屋"为例**：

**Step 1：编写场景描述提示词**

```
一间狭小但整洁的城市出租屋内部全景，约15平米的单间，
靠窗放置一张简易书桌，书桌上有一台笔记本电脑和几本书，
一张单人床铺着深蓝色床单，墙上贴着几张海报，
一个小衣柜，窗户外可以看到城市楼宇的夜景灯光，
木地板，暖黄色台灯光线，夜晚氛围，
日系都市动漫风格，高质量场景设计，无角色，纯场景画面，
宽屏16:9比例，精致的室内细节
```

**Step 2：设置参数**
- 模型：通用v2.0或动漫专用模型
- 比例：16:9（全景场景用横屏更合适）
- 生成数量：4张
- 采样步数：30-35步（场景细节需要更多步数）

**Step 3：生成并选择母版图**
- 生成4张，从场景完整性、细节丰富度、风格一致性三个维度评估
- 选择最佳的一张作为母版图

**Step 4：精修母版图**
- 如果某些细节不满意（如窗户位置不对、书桌太小），使用图生图+低重绘强度（0.3）进行微调
- 母版图的质量直接决定了后续所有变体的质量，值得花时间精修

> 📸 **[截图位置说明：29-01]** 插入生成的场景母版图示例（出租屋全景）。

**Step 5：保存并记录**
- 将母版图保存到场景资产库文件夹
- 创建一个"场景描述.txt"文件，记录：
  ```
  场景名称：主角出租屋
  母版图提示词：[完整提示词]
  使用的模型：通用v2.0
  种子值：[记录种子值]
  关键元素：书桌、笔记本电脑、单人床、窗户、海报、台灯
  光线设定：暖黄色台灯为主光源，窗外城市灯光为辅
  风格：日系都市动漫风格
  ```

### 三、基于母版图批量生成变体

#### 3.1 同一场景不同时间版本

**白天版本**：
- 上传母版图到图生图
- 重绘强度：0.4-0.5
- 提示词：`[母版图描述]，白天，阳光从窗户照入，明亮的室内光线，窗外是蓝天白云`

**黄昏版本**：
- 重绘强度：0.4-0.5
- 提示词：`[母版图描述]，黄昏时刻，橙红色夕阳从窗户照入，温暖的夕阳光线，窗外是晚霞`

**深夜版本**：
- 重绘强度：0.4-0.5
- 提示词：`[母版图描述]，深夜，只有电脑屏幕和台灯的冷白光照亮室内，窗外城市灯火通明`

#### 4.2 场景描述文档模板

每个场景都应有一个描述文档，方便后续生成变体时复用：

```markdown

- 默认光源：台灯（暖黄色）

## 氛围关键词
温暖、狭小、都市、年轻人的出租屋、略带孤独感

- 提示词：[完整提示词]
- 模型：通用v2.0
- 比例：16:9
- 种子值：[记录]

## 变体列表
1. 白天版本 - 阳光照入，明亮温暖

3. 深夜版本 - 只有台灯和屏幕光

```

#### 4.3 场景一致性检查清单

在开始新一集制作前，检查场景资产库：

- [ ] 所有核心场景的母版图是否已创建？
- [ ] 每个场景的变体版本是否足够（至少白天/夜晚两个版本）？
- [ ] 场景描述文档是否完整？
- [ ] 各场景的风格是否统一？
- [ ] 是否需要为新剧情创建新场景？

### 常见问题

**Q1：一个漫剧大概需要多少个场景？**
A：一集1-3分钟的短剧通常需要3-5个场景。一季（10-20集）可能需要10-20个核心场景。建议先建高频场景，按需扩展。

**Q2：场景母版图需要包含角色吗？**
A：母版图最好不要包含角色，保持纯场景。角色在生成具体镜头时通过角色参考功能添加。

**Q3：如何保证不同场景之间的风格统一？**
A：所有场景使用相同的风格描述词和同一模型版本。建议将风格描述词固定下来，在所有场景中复用。

**Q4：场景变体生成后和母版图差异太大怎么办？**
A：降低重绘强度（0.3-0.4），或者在提示词中更详细地描述需要保留的元素。

## 第30课：即梦AI多风格切换——国漫/日漫/赛博朋克/古风参数设置

### 一、主流动漫风格解析

#### 1.1 日系动漫风格

**视觉特征**：
- 大眼睛、精致五官、尖下巴
- 柔和的色彩渐变
- 清晰的轮廓线条
- 头发有光泽感和层次感
- 常见于：校园、恋爱、日常类题材

**核心提示词**：
```
日系动漫风格，精致动漫画风，大而明亮的眼睛，柔和的色彩，
清晰的轮廓线条，anime style，高质量日本动画风格
```

**进阶提示词（参考具体工作室）**：
```
类似京都动画（京阿尼）的画面风格，柔和细腻的色彩过渡，
精致的人物表情刻画，温暖的画面氛围
类似新海诚电影的画面质感，极致的光影效果，通透的天空，
细腻的色彩层次，电影级画面
```

**推荐参数设置**：
| 参数 | 推荐值 |
|------|--------|
| 模型 | 动漫专用模型 或 通用v2.0 |
| 比例 | 9:16（竖屏）或 16:9（横屏） |
| 采样步数 | 25-30步 |
| 引导强度 | 7-8 |

### 二、同一角色多风格切换

**场景**：你已经设计好了角色形象，想看看TA在不同风格下是什么效果。

**操作步骤**：

**Step 1：准备角色基准图**
- 使用之前创建的角色基准图（白色背景的立绘图）

**Step 2：使用图生图+风格迁移**
- 上传角色基准图
- 重绘强度设为0.5-0.6（保留角色特征，改变画风）
- 分别输入不同风格的提示词

**Step 3：对比效果**

| 风格 | 提示词示例（在同一角色基准图上） |
|------|-------------------------------|
| 日漫版 | [角色描述]，日系动漫风格，精致anime画风，大眼睛，柔和色彩 |
| 国漫版 | [角色描述]，国漫风格，国产动漫渲染，东方审美面部特征 |
| 赛博版 | [角色描述]，赛博朋克风格，霓虹灯光映射，科技感装饰 |
| 古风版 | [角色描述]，中国古风仙侠风格，古典仙袍，水墨渲染 |

> 📸 **[截图位置说明：30-05]** 插入同一角色四种风格的对比图（田字格排列）。

**Step 4：选定风格并固化**
- 确定最适合漫剧题材的风格
- 将该风格的关键词固定下来
- 后续所有生成都使用这套风格词

### 三、风格一致性维护

**问题**：漫剧需要保持整部剧的视觉风格统一。

**解决方案**：

1. **建立风格配置文件**：
   ```
   【漫剧风格配置】
   风格名称：都市日漫风
   核心关键词：日系都市动漫风格，柔和的色彩渐变，清晰的轮廓线条，
   精致的人物表情，类似京都动画的画面质感
   模型：通用v2.0
   采样步数：28步
   引导强度：7.5
   比例：9:16
   ```

2. **所有镜头使用同一配置**：每次生成时复制粘贴完全相同的风格词和参数

3. **定期检查**：每生成10-20张后，将所有图片排列在一起，检查是否有风格偏移

### 常见问题

**Q1：如何找到最适合自己漫剧的风格？**
A：先用同一组角色和场景描述，分别尝试3-5种不同风格，对比效果后选择最满意的。也可以混合风格，如"日系动漫+赛博朋克元素"。

**Q2：风格关键词太多会不会互相冲突？**
A：会。建议核心风格词不超过3-4个，辅助词2-3个。太多关键词会让AI困惑。

**Q3：为什么我用了赛博朋克关键词但画面不够赛博朋克？**
A：赛博朋克风格需要场景和光线的配合。除了风格词，还要在场景描述中加入霓虹灯、夜景、雨天等元素。

**Q4：可以混合两种风格吗？**
A：可以，但要注意兼容性。"日漫+赛博朋克"可以融合，但"日漫+水墨"可能效果不佳。多试几次找到最佳组合。

### 课后练习

1. **风格实验**：用同一角色描述，分别生成日漫、国漫、赛博朋克、古风四种风格的图片
2. **风格混合**：尝试创造一种独特的混合风格（如"水墨赛博朋克"），记录效果
3. **风格配置**：为你的漫剧建立一份完整的风格配置文件
4. **一致性测试**：用你的风格配置连续生成10张图片，检查风格是否一致

## 第31课：即梦AI与剪映生态整合——创作-剪辑-分发流程

### 学习目标

1. 理解即梦AI与剪映的生态整合优势
2. 掌握从即梦AI到剪映的完整工作流
3. 学会在剪映中对AI生成内容进行二次加工
4. 了解从剪辑到多平台分发的流程

### 一、即梦AI+剪映的生态优势

**为什么是"黄金组合"？**

| 优势 | 说明 |
|------|------|
| 同属字节系 | 数据互通，账号打通，素材可直接流转 |
| 即梦负责"生" | AI生成关键帧画面和动态视频 |
| 剪映负责"剪" | 拼接、配音、字幕、特效、转场 |
| 全流程闭环 | 从创意到发布，一个生态搞定 |

**工作流总览**：
```
即梦AI（生成画面） → 导出图片/视频 → 导入剪映 → 剪辑加工 → 导出成片 → 分发到各平台
     ↑                                              ↓
     └──── 剪映中发现问题，回到即梦AI修改 ←──────────┘
```

### 二、从即梦AI导出素材

1. 在即梦AI中找到你生成的图片（"我的作品"页面）
2. 点击图片放大预览
3. 点击右上角的**"下载"按钮**（向下箭头图标）
4. 选择分辨率（通常选最高分辨率）
5. 图片会保存到本地（通常在"下载"文件夹中）

#### 3.2 导入素材

**操作步骤**：
1. 点击剪映左上角的**"导入"按钮**
2. 选择"导入素材"
3. 找到你保存即梦AI素材的文件夹

#### 3.3 将关键帧图片转为视频

**方法一：直接添加到时间线**
1. 将关键帧图片从素材库拖入时间线
2. 默认每张图片显示3秒
3. 调整显示时长：点击时间线上的图片片段，拖动边缘调整时长
4. **漫剧建议**：每个关键帧显示2-4秒（根据旁白/对白节奏调整）

**方法二：添加Ken Burns效果（画面缓慢移动）**
1. 选中时间线上的图片片段
2. 点击右侧面板的**"动画"**选项
3. 选择"入场动画"或"出场动画"
4. 选择"缓慢推进""缓慢左移"等效果
5. 这会让静态图片有动态感，比纯静态更有观看体验

**方法三：使用即梦AI的图生视频**

#### 3.4 添加配音和对白

**使用剪映AI配音**：
1. 点击剪映的**"文本"**选项卡
2. 选择**"AI配音"**或"文本朗读"
3. 输入对白/旁白文字
4. 选择AI声音（男声/女声、年轻/成熟、温柔/有力等）
5. 点击"生成"，AI配音会自动添加到时间线
6. 调整配音与画面的同步

**旁白脚本示例**：
```
旁白：那一年，林晓25岁，刚刚毕业一年。
（对应画面：主角坐在出租屋电脑前的中景）

旁白：每天重复着写代码、加班、回到出租屋的生活。
（对应画面：主角疲惫地推开出租屋的门）

林晓（自言自语）：这就是我想要的生活吗？
（对应画面：主角看着窗外城市夜景的特写）
```

#### 3.5 添加背景音乐和音效

**添加背景音乐**：
1. 点击**"音频"**选项卡
2. 选择**"音乐库"**
3. 浏览或搜索合适的BGM
4. 点击"+"添加到时间线
5. 调整音量（建议BGM音量为旁白的30%-50%）

**添加音效**：
1. 在"音频"选项卡中选择**"音效"**
2. 搜索需要的音效（脚步声、风声、键盘声、雨声等）
3. 将音效拖入时间线，对齐对应的画面

**漫剧常用音效**：
- 转场音效：whoosh、叮铃
- 环境音效：城市噪音、风声、雨声、鸟鸣
- 情绪音效：心跳声（紧张时）、钟表声（孤独时）
- 动作音效：脚步声、开门声、键盘敲击声

#### 3.6 添加字幕

**方法一：自动识别字幕**
1. 如果已有配音，点击**"文本"→"智能字幕"→"识别字幕"**
2. 剪映会自动将语音转为文字字幕
3. 检查并修正识别错误

**方法二：手动添加字幕**
1. 点击**"文本"→"新建文本"**
2. 输入字幕文字
3. 设置字体、大小、颜色、位置
4. 在时间线上调整字幕的显示时间

**漫剧字幕建议**：
- 对白字幕放在画面下方中间
- 旁白字幕可以放在画面下方或使用不同样式（如斜体）
- 角色名字幕可以使用不同颜色区分

#### 3.7 添加转场效果

**操作步骤**：
1. 在时间线上两个片段之间找到**转场标记**
2. 点击转场标记，弹出转场效果库
3. 选择合适的转场效果：
   - 场景切换：淡入淡出、黑场过渡
   - 快节奏：闪白、模糊过渡
   - 时间流逝：时钟旋转、日历翻页
4. 设置转场时长（建议0.5-1秒）

**漫剧转场建议**：
- 不要过度使用花哨的转场
- 同一集内保持转场风格统一
- 重要剧情转折用"黑场+淡入"增强效果

### 二、局部重绘功能

#### 2.1 什么是局部重绘

局部重绘（Inpainting）允许你只修改图片的某一部分，而保持其余部分不变。这是修复AI缺陷和微调画面的最精准工具。

**应用场景**：
- 修复手部变形
- 修改面部表情
- 更换服装颜色/样式
- 添加/移除画面元素
- 修复背景瑕疵

#### 2.2 局部重绘操作步骤

**Step 1：生成基础图片**
- 先用文生图或图生图生成一张基础图片
- 这张图大部分内容满意，只有局部需要修改

**Step 2：进入局部重绘模式**
- 点击图片下方的**"编辑"按钮**
- 在弹出的编辑菜单中选择**"局部重绘"**

> 📸 **[截图位置说明：33-04]** 插入"编辑"按钮和"局部重绘"选项的位置截图。

**Step 3：绘制蒙版（Mask）**
- 进入局部重绘界面后，你会看到原图和一个画笔工具
- **用画笔涂抹你想要修改的区域**
- 涂抹的区域（通常显示为红色/半透明覆盖）就是AI会重新生成的部分
- 未涂抹的区域将保持不变

**画笔设置**：
- **画笔大小**：根据需要修改的区域大小调节
  - 小瑕疵（如手指）：小画笔
  - 大区域（如整个背景）：大画笔
- **画笔硬度**：建议使用中等硬度
  - 硬边缘：修改区域边界清晰
  - 软边缘：修改区域与周围自然过渡（推荐）

**绘制技巧**：
- 涂抹范围比需要修改的区域稍微大一圈
- 这样AI在生成新内容时有足够的"过渡空间"
- 避免涂抹太小的区域，AI可能无法生成连贯的内容

> 📸 **[截图位置说明：33-05]** 插入局部重绘蒙版绘制界面截图，展示画笔涂抹区域。

**Step 4：输入重绘提示词**
- 在提示词框中描述你希望该区域变成什么
- **只描述被涂抹区域的内容，不要描述整个画面**

**示例**：
- 修改手部：`正常的人手，五根手指，自然的手部姿势，手握着咖啡杯`
- 修改表情：`温柔的微笑表情，明亮的眼睛，愉悦的面部表情`
- 修改服装：`穿着深蓝色西装外套，白色衬衫内搭，正式商务着装`
- 修改背景：`夜晚的城市街道，霓虹灯招牌，雨后的湿润路面`

**Step 5：设置重绘参数**

**局部重绘专用参数**：
| 参数 | 建议值 | 说明 |
|------|--------|------|
| 重绘强度 | 0.7-0.9 | 局部重绘通常需要较高强度，因为是完全重新生成该区域 |
| 采样步数 | 25-30 | 与正常生成相同 |
| 引导强度 | 7-8 | 与正常生成相同 |

**Step 6：生成并检查**
- 点击"生成"
- AI只会在涂抹区域重新生成内容，其他部分保持不变
- 检查新生成的部分是否与周围自然融合
- 如果不满意，调整蒙版范围或提示词，重新生成

> 📸 **[截图位置说明：33-06]** 插入局部重绘前后对比图，左边是原图（有缺陷），右边是修复后的图。

#### 2.3 局部重绘进阶技巧

**技巧1：多次迭代修复**
- 第一次重绘可能不完美
- 可以在第一次重绘结果的基础上再次进行局部重绘
- 每次只解决一个小问题，逐步逼近完美

**技巧2：边缘融合处理**
- 如果重绘区域与周围有明显接缝
- 可以扩大蒙版范围，让AI有更多"过渡空间"
- 或者降低重绘强度（0.6-0.7），让过渡更自然

**技巧3：颜色引导**
- 在重绘提示词中明确指定颜色
- 示例：`穿着与原图相同颜色的深灰色卫衣`
- 这有助于保持与未修改区域的色彩一致

**技巧4：批量修复**
- 如果多个镜头都有类似的缺陷（如手部问题）
- 可以建立一个"修复提示词模板"
- 每次只需要涂抹+粘贴模板提示词

### 三、其他高级功能

#### 3.1 智能扩图

**功能**：在原图基础上向外扩展画面内容

**使用场景**：
- 竖屏图需要变成横屏图
- 画面构图太紧凑，需要更多背景空间
- 需要将特写扩展为中景

**操作步骤**：
1. 选择图片 → 点击"编辑"→"扩图"
2. 选择目标比例（如从1:1扩展到16:9）
3. AI会自动向外填充内容
4. 检查扩展部分是否自然

#### 3.2 去背景/换背景

**功能**：自动去除图片背景或替换为新背景

**操作步骤**：
1. 选择图片 → 点击"编辑"→"去背景"
2. AI会自动识别主体并去除背景
3. 可以选择透明背景或纯色背景
4. 也可以选择"换背景"，输入新背景描述

#### 3.3 画面超分辨率

**功能**：将低分辨率图片放大为高分辨率

**使用场景**：
- 即梦AI默认生成的图片分辨率不够高
- 需要打印或大屏展示

**操作步骤**：
1. 选择图片 → 点击"超分辨率"或"高清放大"
2. 选择放大倍数（2x/4x）
3. AI会智能放大并补充细节

### 四、高级功能组合实战

**场景**：一张漫剧关键帧画面存在多个问题

**原图问题**：
1. 手部有轻微变形
2. 表情不够生动（应该是微笑但看起来平淡）
3. 背景不够丰富
4. 分辨率偏低

**修复流程**：

```
Step 1：局部重绘手部
    → 涂抹手部区域，提示词："正常的人手，自然握着杯子"
    → 生成，选择最佳结果
    
Step 2：局部重绘面部
    → 涂抹面部区域，提示词："温柔的微笑，明亮有神的眼睛"
    → 生成，选择最佳结果
    
Step 3：智能扩图
    → 将画面从特写扩展为中景
    → 生成，检查扩展效果
    
Step 4：超分辨率
    → 放大到2x
    → 获得高清最终版本
```

### 常见问题

**Q1：局部重绘后和原图风格不一致怎么办？**
A：在重绘提示词中加入与原图相同的风格描述词。或者使用图生图+低重绘强度对整个画面进行风格统一。

**Q2：多图参考最多能用几张？**
A：目前通常支持2-4张参考图。具体数量可能随版本更新变化。

**Q3：局部重绘的蒙版画不好怎么办？**
A：可以放大图片后再画蒙版，这样更精确。也可以使用"智能选区"功能（如果可用），AI会自动识别物体边界。

**Q4：扩图后边缘有接缝怎么办？**
A：可以对整个画面用图生图+低重绘强度（0.2-0.3）过一遍，让AI统一处理边缘过渡。

### 五、Step 4：质量检查与精修

#### 5.1 一致性检查

将所有13张关键帧排列在一起，检查：

- [ ] **角色一致性**：林晓在所有镜头中是否像同一个人？
  - 发型、发色是否一致？
  - 服装是否一致？
  - 面部特征是否相似？
- [ ] **场景一致性**：同一场景的不同镜头是否连贯？
  - 出租屋的3个镜头（001-003）场景是否一致？
  - 天台的5个镜头（009-013）场景是否一致？
- [ ] **风格一致性**：所有画面是否使用相同的画风？
  - 色调是否统一？
  - 线条风格是否一致？
- [ ] **情绪一致性**：画面情绪是否与剧本匹配？
  - 镜头001-003是否传达出疲惫感？
  - 镜头007-008是否传达出紧张冲突？
  - 镜头009-013是否传达出从落寞到希望的转变？

#### 5.2 精修操作

**对不满意的画面进行精修**：

**方法1：局部重绘**
- 手部有问题 → 局部重绘手部
- 表情不对 → 局部重绘面部
- 服装细节缺失 → 局部重绘服装区域

**方法2：图生图微调**
- 整体风格偏了 → 图生图+低重绘强度（0.3）统一风格
- 角色不够像 → 图生图+中重绘强度（0.5）+角色参考

**方法3：重新生成**
- 如果某张图完全不满意，直接用修改后的提示词重新生成
- 记录之前生成的种子值，微调提示词时固定种子

### 常见问题

**Q1：一集漫剧需要多少个关键帧？**
A：1分钟的短剧通常需要10-20个关键帧。具体取决于叙事节奏——对话密集的场景需要更多镜头切换，抒情场景可以用更少的镜头。

**Q2：关键帧生成花了太长时间怎么办？**
A：熟练后可以大幅提速：建立角色和场景资产库后，后续镜头只需要复制粘贴+微调提示词。预计第二集开始，生成时间可以缩短50%以上。

**Q3：多角色同框画面总是不好怎么办？**
A：多角色同框是AI的难点。建议：(1)先生成单角色场景再用图生图添加另一角色；(2)使用远景减少面部细节要求；(3)用剪映后期合成（生成两张分别包含不同角色的图，用剪映叠加）。

**Q4：如何保持后续集数的风格一致？**
A：(1)使用本集建立的角色基准图和场景母版图；(2)固定风格描述词和参数设置；(3)定期将新画面与之前的画面放在一起对比检查。
