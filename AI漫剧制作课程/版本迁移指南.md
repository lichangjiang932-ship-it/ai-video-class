## 如何获取最新版本

### Git 更新

```bash
# 如果你已经克隆了仓库
cd ai-video-class
git pull origin main

# 如果是首次克隆
git clone https://github.com/lichangjiang932-ship-it/ai-video-class.git
```

### 手动下载

1. 访问 GitHub 仓库页面
2. 点击 "Code" → "Download ZIP"
3. 解压后覆盖旧文件

## 常见问题

**Q：更新后旧文件会被覆盖吗？**
A：不会。所有更新都是新增或补充，不会删除已有内容。

**Q：我需要重新学习吗？**
A：不需要。新增内容是补充，不影响已完成的学习。

**Q：PPT课件有变化吗？**
A：PPT文件本身没有变化，但新增了Python数据源文件，方便自定义。

**Q：如何查看我当前的版本？**
A：查看 `CHANGELOG.md` 文件，对比最新版本号。
