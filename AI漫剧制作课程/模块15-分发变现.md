### 105.1 算法逻辑

#### 105.1.1 推荐算法核心指标

| 指标 | 权重 | 说明 | 优化方向 |
|------|------|------|---------|
| **完播率** | ★★★★★ | 最重要，前3秒决定 | 优化开头钩子 |
| **互动率** | ★★★★☆ | 点赞、评论、分享 | 增加互动引导 |
| **关注率** | ★★★☆☆ | 看完后关注 | 优化结尾钩子 |
| **停留时长** | ★★★☆☆ | 在作品上花的时间 | 增加内容密度 |
| **转发率** | ★★☆☆☆ | 分享给他人 | 增加社交价值 |

#### 105.1.2 推荐流量池机制

```
流量池递进机制

初始池：200-500播放
├─ 完播率>30% + 互动率>5% → 进入下一级
│
二级池：1000-5000播放
├─ 数据持续优秀 → 进入下一级
│
三级池：1万-10万播放
├─ 爆款潜力 → 进入下一级
│
四级池：10万-100万播放
│
五级池：100万+播放（爆款）
```

#### 105.1.3 前3秒法则

```
前3秒决定生死：

第1秒：视觉冲击
├─ 震撼画面/冲突场景/悬念钩子
├─ 必须在1秒内抓住注意力
└─ 避免缓慢开场

第2秒：情绪建立
├─ 呈现核心冲突/矛盾
├─ 让用户产生情感共鸣
└─ 激发好奇心

第3秒：信息传达
├─ 告诉用户"这是什么故事"
├─ 建立观看预期
└─ 留下继续观看的理由
```

### 105.3 标题技巧

```

**Q1：新账号没有流量怎么办？**

**Q3：如何从抖音获取收入？**
A：(1) 中视频计划（播放量分账）；(2) 星图广告（品牌合作）；(3) 直播带货；(4) 付费短剧。

## 第106课：红果短剧分账机制详解

### 学习目标

1. 理解红果的分账模式和收益计算
2. 掌握提升作品评级的方法
3. 学会优化红果平台的运营策略

---

### 106.1 分账模式

#### 106.1.1 收益计算公式

```
基础分账收益 = 有效播放量 × 单价 × 分成系数 × 评级系数

影响因素：
- 有效播放量：去除刷量后的实际播放
- 单价：每个有效播放的单价（CPM）
- 分成系数：创作者与平台的分成比例
- 评级系数：S/A/B/C不同评级的系数
```

#### 106.1.2 收益参考

| 评级 | 分成系数 | 单集预估收益（10万播放） |
|------|---------|----------------------|
| S级 | 1.0 | 3000-5000元 |
| A级 | 0.7 | 2000-3500元 |
| B级 | 0.5 | 1500-2500元 |
| C级 | 0.3 | 800-1500元 |

---

### 106.2 评级体系

#### 106.2.1 评级标准

| 评级 | 完播率 | 互动率 | 更新频率 | 内容质量 |
|------|--------|--------|---------|---------|
| **S级** | >50% | >8% | 日更 | 优秀 |
| **A级** | >40% | >5% | 隔日更 | 良好 |
| **B级** | >30% | >3% | 周3更 | 中等 |
| **C级** | <30% | <3% | 不稳定 | 一般 |

#### 106.2.2 提升评级的方法

```
评级提升策略：

1. 提高完播率
- 优化前3秒钩子
- 增加爽点密度
- 控制每集时长（3-5分钟）
- 每集结尾留钩子

2. 提高互动率
- 在视频中设置互动引导
- 在评论区引导讨论
- 回复用户评论
- 设置投票/问答

3. 保持更新频率
- 制定更新计划
- 提前储备内容
- 固定时间发布
- 保持连续性

4. 提升内容质量
- 画质达到1080P
- 音画同步准确
- 角色一致性好
- 剧情逻辑通顺
```

---

### 106.3 红果运营策略

#### 106.3.1 内容策略

```
红果用户偏好：
- 玄幻/仙侠（最大品类）
- 都市逆袭（增长最快）
- 甜宠/言情（女性用户）
- 悬疑/推理（高粘性）

内容特点：
- 节奏快，爽点密集
- 每集有明确钩子
- 角色形象鲜明
- 画面质量高
```

#### 106.3.2 发布策略

```
红果发布要点：
- 一次性发布3-5集（建立追剧习惯）
- 之后保持日更或隔日更
- 每集时长3-5分钟
- 每集结尾必须有钩子
```

---

### 106.4 常见问题

**Q1：红果和抖音有什么区别？**
A：红果是付费短剧平台，用户付费意愿更强；抖音是免费+广告模式，流量更大。建议两个平台同时运营。

**Q2：如何入驻红果？**
A：(1) 注册红果创作者账号；(2) 提交作品审核；(3) 审核通过后发布；(4) 达到条件后开通分账。

**Q3：红果的审核标准是什么？**
A：(1) 内容合规；(2) 画质达标；(3) AIGC标识完整；(4) 无侵权内容。

- 中间太拖沓 → 增加爽点密度，缩短时长
- 结尾没悬念 → 加强结尾钩子

- 在视频中加入互动引导（"你觉得...？"）
- 在评论区引导讨论
- 设置投票/问答

关注率低（<2%）→ 优化留存：
- 增加系列感（"第X集"标注）
- 在结尾预告下集内容
- 建立角色品牌认知

某平台数据特别差 → 分析原因：
- 是内容不适合该平台用户？
- 是发布时间不对？
- 是封面/标题不够吸引人？
```

隔日更（可接受）：

- 保持稳定节奏
- 适合品质优先的创作者

周更（仅限精品）：

- 品质最高

- 宁可降低频率，也不要降低品质

### 112.5 常见问题

**Q1：多平台运营精力不够怎么办？**
A：(1) 使用一键分发工具；(2) 集中精力主攻1-2个平台；(3) 建立团队分工；(4) 使用自动化工具。

**Q2：不同平台的内容需要完全不一样吗？**
A：核心内容可以相同，但封面、标题、简介需要针对各平台调整。节奏和时长也可以微调。

**Q3：如何判断哪个平台最适合我？**
A：(1) 在各平台同步发布测试；(2) 对比2-4周的数据；(3) 选择数据最好的平台重点投入。
