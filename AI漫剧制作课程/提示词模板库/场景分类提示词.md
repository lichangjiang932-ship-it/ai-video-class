## 五、负向提示词

### 通用负向提示词
```
低质量，模糊，变形，扭曲，多余肢体，多余手指，
水印，文字，logo，签名，边框，画面裁切不完整，
过曝，过暗，噪点，jpeg压缩痕迹，
3D渲染感，CG感，不自然的光泽
```

### 角色负向提示词
```
面部变形，五官不协调，大小眼，歪嘴，
手指数量错误，手指扭曲，肢体比例失调，
多余的手臂，多余的腿，身体部位重叠，
服装不一致，发型不一致，面部特征变化
```

```

动作僵硬，不自然的运动，物理效果错误，
画质下降，帧率不稳，音画不同步
```

## 十、使用建议

1. **模板不是终点**：根据你的具体角色/场景修改模板中的细节
2. **组合使用**：角色描述 + 场景描述 + 镜头类型 + 风格描述 = 完整提示词
3. **负向提示词必加**：能有效减少生成瑕疵
4. **参考图配合**：提示词 + 参考图效果远好于纯提示词
5. **迭代优化**：首次生成不理想时，调整提示词后重新生成
6. **视频提示词要简化**：视频生成的提示词比图像更简洁，避免过于复杂
7. **运镜要明确**：视频提示词中明确指定镜头运动方式
8. **动作要单一**：一个视频片段只描述一个主要动作
