# 📏 质量标准图鉴 — AI宠物拟人漫剧

> 各维度的合格/良好/优秀标准说明，用金猫搭档做示例。

---

## 评分体系（总分135分）

| 维度 | 分值 | 权重 |
|------|------|------|
| 画面质量 | 40分 | 30% |
| 音频质量 | 25分 | 20% |
| 叙事质量 | 40分 | 30% |
| 后期质量 | 30分 | 20% |

---

## 一、画面质量（40分）

### 1.1 角色一致性（15分）

| 等级 | 标准 | 金猫搭档示例 |
|------|------|-------------|
| 优秀（12-15分） | 跨镜头角色外貌高度一致 | 豆包在所有镜头中毛色、脸型、耳朵特征一致 |
| 良好（8-11分） | 偶尔有轻微差异但不影响辨识 | 豆包毛色偶尔偏深/偏浅，但整体特征一致 |
| 合格（4-7分） | 有明显差异但能认出是同一角色 | 豆包脸型偶尔变化，但金毛特征保持 |
| 不合格（0-3分） | 跨镜头角色完全不像 | 不同镜头看起来像不同的狗 |

### 1.2 毛发纹理（10分）

| 等级 | 标准 |
|------|------|
| 优秀（8-10分） | 毛发纹理清晰自然，有层次感和光泽 |
| 良好（5-7分） | 毛发纹理基本清晰，偶有模糊 |
| 合格（3-4分） | 毛发纹理存在但不够细腻 |
| 不合格（0-2分） | 毛发模糊或过度磨皮变成"塑料感" |

### 1.3 画面构图（10分）

| 等级 | 标准 |
|------|------|
| 优秀（8-10分） | 构图美观，角色位置合理，背景干净 |
| 良好（5-7分） | 构图基本合理，偶有遮挡或偏移 |
| 合格（3-4分） | 构图能用但不够美观 |
| 不合格（0-2分） | 构图混乱，角色被截断或遮挡 |

### 1.4 调色风格（5分）

| 等级 | 标准 |
|------|------|
| 优秀（5分） | 色调统一，电影质感明显，温馨舒适 |
| 良好（3-4分） | 色调基本统一，偶有偏差 |
| 合格（1-2分） | 有色调整体感但不够精致 |
| 不合格（0分） | 色调混乱，无整体感 |

---

## 二、音频质量（25分）

### 2.1 配音自然度（10分）

| 等级 | 标准 | 金猫搭档示例 |
|------|------|-------------|
| 优秀（8-10分） | 声音自然，情绪到位，节奏感好 | 豆包声音温暖带紧张，奶盖声音冷淡不耐烦 |
| 良好（5-7分） | 声音基本自然，情绪基本到位 | 声音能听但偶尔机械感 |
| 合格（3-4分） | 能听清内容但不够自然 | 配音像机器人朗读 |
| 不合格（0-2分） | 听不清或完全机械 |

### 2.2 音效精准度（8分）

| 等级 | 标准 |
|------|------|
| 优秀（7-8分） | 音效与画面动作精确同步，增强表现力 |
| 良好（4-6分） | 音效基本同步，偶有延迟 |
| 合格（2-3分） | 有音效但时机不够精准 |
| 不合格（0-1分） | 无音效或音效完全不搭 |

### 2.3 BGM配合（7分）

| 等级 | 标准 |
|------|------|
| 优秀（6-7分） | BGM与情绪完美配合，音量平衡恰当 |
| 良好（4-5分） | BGM基本合适，偶有音量失衡 |
| 合格（2-3分） | 有BGM但与内容不够搭配 |
| 不合格（0-1分） | 无BGM或BGM喧宾夺主 |

---

## 三、叙事质量（40分）

### 3.1 对话节奏（15分）

| 等级 | 标准 | 金猫搭档示例 |
|------|------|-------------|
| 优秀（12-15分） | 节奏感强，起承转合清晰，欲罢不能 | 豆包热情→奶盖冷淡→吐槽→温馨收尾 |
| 良好（8-11分） | 节奏基本流畅，偶有拖沓或过快 | 对话流畅但个别地方节奏不均 |
| 合格（4-7分） | 能看下去但节奏平淡 | 全程一个节奏，无起伏 |
| 不合格（0-3分） | 节奏混乱，看不下去 |

### 3.2 情绪表达（15分）

| 等级 | 标准 |
|------|------|
| 优秀（12-15分） | 情绪层次丰富，观众能感受到角色的情感 |
| 良好（8-11分） | 情绪基本到位，偶有平淡 |
| 合格（4-7分） | 有情绪但不够鲜明 |
| 不合格（0-3分） | 无情绪或情绪混乱 |

### 3.3 笑点/亮点密度（10分）

| 等级 | 标准 |
|------|------|
| 优秀（8-10分） | 每15s至少一个笑点或亮点 |
| 良好（5-7分） | 每30s有一个笑点或亮点 |
| 合格（3-4分） | 偶尔有笑点 |
| 不合格（0-2分） | 全程无笑点 |

## 四、后期质量（30分）

### 4.1 磨皮滤镜（10分）

| 等级 | 标准 |
|------|------|
| 优秀（8-10分） | 轻微磨皮，保留毛发纹理，电影质感 |
| 良好（5-7分） | 磨皮适度，毛发基本保留 |
| 合格（3-4分） | 有磨皮但过度或不足 |
| 不合格（0-2分） | 过度磨皮（塑料感）或无磨皮 |

### 4.2 声波特效（10分）

| 等级 | 标准 |
|------|------|
| 优秀（8-10分） | 声波特效与配音同步，增强表现力 |
| 良好（5-7分） | 有声波特效，基本同步 |
| 合格（3-4分） | 有特效但不够精致 |
| 不合格（0-2分） | 无特效或特效花哨 |

### 4.3 字幕设计（5分）

| 等级 | 标准 |
|------|------|
| 优秀（5分） | 字幕清晰，角色区分明确，位置合理 |
| 良好（3-4分） | 字幕清晰，基本合理 |
| 合格（1-2分） | 有字幕但不够规范 |
| 不合格（0分） | 无字幕或字幕遮挡画面 |

### 4.4 转场效果（5分）

| 等级 | 标准 |
|------|------|
| 优秀（5分） | 转场自然流畅，增强节奏感 |
| 良好（3-4分） | 转场基本自然 |
| 合格（1-2分） | 有转场但偶有突兀 |
| 不合格（0分） | 无转场或转场花哨 |
