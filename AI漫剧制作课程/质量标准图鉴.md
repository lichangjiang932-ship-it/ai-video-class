# 📏 质量标准图鉴

> AI漫剧的质量评判标准，含各维度的合格/良好/优秀标准说明。

---

## 一、质量评估维度

### 总览

| 维度 | 权重 | 合格（B） | 良好（A） | 优秀（S） |
|------|------|---------|---------|---------|
| **角色一致性** | 25% | 80-90% | 90-95% | >95% |
| **画面质量** | 20% | 720P，有小瑕疵 | 1080P，基本无瑕 | 1080P+，精美 |
| **动作自然度** | 15% | 基本可接受 | 流畅自然 | 物理真实 |
| **音频质量** | 15% | 发音准确 | 有情感 | 情感到位+音效丰富 |
| **节奏感** | 15% | 有爽点 | 爽点密集 | 张弛有度+钩子强 |
| **合规性** | 10% | 基本合规 | 完全合规 | 合规+主动标识 |

## 二、角色一致性标准

### 合格标准（80-90%）

```
✅ 面部基本特征一致（五官位置、比例）
✅ 发型颜色和大致造型一致
✅ 服装风格基本一致
❌ 偶尔出现细节变化（耳环、配饰等）
❌ 不同角度时辨识度下降
```

### 良好标准（90-95%）

```
✅ 面部特征高度一致
✅ 发型、服装、配饰全部一致
✅ 不同角度可辨认
✅ 跨场景保持一致
❌ 极端角度偶尔偏差
```

### 优秀标准（>95%）

```
✅ 面部特征完全一致
✅ 所有细节一致（包括微小配饰）
✅ 任何角度都可辨认
✅ 跨集数保持一致
✅ 表情变化时特征不漂移
```

### 提升方法

| 方法 | 效果 | 难度 |
|------|------|------|
| 固定参考图 | ★★★☆☆ | ★☆☆☆☆ |
| IP-Adapter | ★★★★☆ | ★★☆☆☆ |
| LoRA训练 | ★★★★★ | ★★★★☆ |
| 角色档案（Seedance） | ★★★★☆ | ★★☆☆☆ |

## 三、画面质量标准

### 合格标准

```
✅ 分辨率达到720P以上
✅ 构图基本合理，主体突出
✅ 色彩基本协调
❌ 偶尔出现轻微变形
❌ 部分画面有噪点或模糊
```

### 良好标准

```
✅ 分辨率达到1080P
✅ 构图合理，有景别变化
✅ 色调统一，光影自然
✅ 无明显变形或瑕疵
✅ 细节清晰
```

### 优秀标准

```
✅ 分辨率达到1080P以上
✅ 构图精美，有电影感
✅ 色调统一，光影考究
✅ 无任何瑕疵
✅ 细节丰富，质感真实
```

### 常见问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 面部变形 | 提示词不准/模型问题 | 负向提示词+ADetailer |
| 手指异常 | AI通病 | 负向提示词+后期修复 |
| 色调不统一 | 不同生成批次 | 统一调色预设 |
| 画面模糊 | 分辨率低/压缩 | 提高分辨率+码率 |
| 物体穿模 | AI理解错误 | 调整提示词+重生成 |

## 四、动作自然度标准

### 合格标准

```
✅ 基本动作可接受（走路、站立、说话）
✅ 无严重变形或闪烁
❌ 复杂动作不自然
❌ 物理效果有违和感
```

### 良好标准

```
✅ 常见动作流畅自然
✅ 物理效果基本真实
✅ 无明显变形或闪烁
✅ 打斗场景可接受
❌ 极端动作偶尔失真
```

### 优秀标准

```
✅ 所有动作流畅自然
✅ 物理效果完全真实
✅ 打斗场景有冲击力
✅ 表情动作细腻
✅ 头发/衣物飘动自然
```

### 工具选择建议

| 场景 | 推荐工具 | 原因 |
|------|---------|------|
| 对话/口型 | Vidu Q3 | 音画同步最佳 |
| 拳脚打斗 | 可灵AI | 物理仿真最强 |
| 法术/特效 | 可灵AI | 特效表现力好 |
| 慢动作 | 海螺AI | 细节保持好 |
| 走路/跑步 | Seedance 2.0 | 动作参考驱动 |

## 五、音频质量标准

### 配音标准

| 等级 | 发音准确性 | 情感表达 | 语速节奏 | 口型同步 |
|------|-----------|---------|---------|---------|
| 合格 | >95% | 基本到位 | 正常 | 基本对齐 |
| 良好 | >98% | 有情感变化 | 自然 | 对齐 |
| 优秀 | 100% | 情感饱满 | 张弛有度 | 完美对齐 |

### BGM标准

| 等级 | 风格匹配 | 音量平衡 | 节奏配合 | 过渡自然 |
|------|---------|---------|---------|---------|
| 合格 | 基本匹配 | 不盖过对白 | 基本配合 | 有 |
| 良好 | 匹配 | 平衡 | 配合 | 自然 |
| 优秀 | 完美匹配 | 精确平衡 | 紧密配合 | 无缝过渡 |

### 音量平衡参考值

```
对白配音：0dB（基准）
BGM：-12dB ~ -18dB
音效：-6dB ~ -12dB
环境音：-18dB ~ -24dB
```

## 六、节奏感标准

### 每集节奏模板

```
00:00-00:03  钩子（视觉冲击/冲突/悬念）—— 必须有
00:03-01:00  铺垫（建立矛盾/压力）
01:00-01:30  小爽点（初步反击/突破）—— 建议有
01:30-03:00  升级（矛盾升级/新挑战）
03:00-04:30  大爽点（核心爽点爆发）—— 必须有
04:30-05:00  钩子结尾（新悬念/下集预告）—— 必须有
```

### 节奏评估

| 等级 | 前3秒钩子 | 爽点密度 | 结尾钩子 | 整体节奏 |
|------|---------|---------|---------|---------|
| 合格 | 有 | 每集1个 | 有 | 基本流畅 |
| 良好 | 强 | 每集2-3个 | 强 | 张弛有度 |
| 优秀 | 极强 | 每集3-5个 | 极强 | 电影级节奏 |
