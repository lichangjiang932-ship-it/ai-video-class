## 阶段一：前期策划

### 1.1 选题与定位
- [ ] 题材已确定：____________________
- [ ] 目标受众已明确
- [ ] 核心爽点已确定：____________________
- [ ] 与同题材竞品的差异化已分析

### 1.2 剧本
- [ ] 本集大纲已完成（100字以内）
- [ ] 完整剧本已完成（600-800字）
- [ ] 前3秒有强钩子（冲突/悬念/反转）
- [ ] 结尾有悬念/钩子
- [ ] 对白口语化、有网感
- [ ] 场景描述可视化、适合AI生成
- [ ] 已用AI检查剧本逻辑漏洞

### 1.3 分镜
- [ ] 分镜脚本已完成（15-25个镜头）
- [ ] 每个镜头标注了景别（特写/中景/远景）
- [ ] 每个镜头标注了AI提示词
- [ ] 景别变化有节奏（不是全部中景）
- [ ] 关键情绪点有特写镜头
- [ ] 场景转换有过渡设计

## 阶段二：角色与场景资产

### 2.1 角色资产
- [ ] 主角基准图已完成
- [ ] 主角多角度参考图已完成（正面/侧面/背面）
- [ ] 主角多表情参考图已完成（平静/微笑/愤怒/悲伤）
- [ ] 配角基准图已完成
- [ ] 角色外貌锚定描述已记录
- [ ] 角色档案已上传到AI工具

### 2.2 场景资产

  - [ ] 场景1：____________________
  - [ ] 场景2：____________________
  - [ ] 场景3：____________________
- [ ] 每个场景有白天/夜晚两个版本
- [ ] 场景描述文档已记录
- [ ] 风格配置已统一

---

## 阶段三：图像生成

### 3.1 关键帧生成
- [ ] 全部关键帧已生成（____张）
- [ ] 每个镜头从4个候选中选了最佳的
- [ ] 角色在所有帧中外观一致
- [ ] 场景风格在所有帧中统一
- [ ] 光影方向在所有帧中一致
- [ ] 无明显AI伪影（手指/面部/文字）

### 3.2 质量检查
- [ ] 已将所有关键帧排列在一起对比检查
- [ ] 角色一致性评分：____/5
- [ ] 场景一致性评分：____/5
- [ ] 整体画质评分：____/5
- [ ] 不满意的帧已重新生成或修复

---

## 阶段四：视频生成

### 4.1 视频片段生成
- [ ] 全部视频片段已生成（____段）
- [ ] 每段视频从4个候选中选了最佳的
- [ ] 运动幅度合适（不过大/不过小）
- [ ] 动作自然流畅
- [ ] 无明显变形或闪烁

### 4.2 连续性检查
- [ ] 使用首尾帧控制确保片段衔接
- [ ] 片段之间的过渡自然
- [ ] 动作逻辑连贯
- [ ] 无跳帧或突变

### 4.3 运镜检查
- [ ] 运镜方式符合分镜设计
- [ ] 运镜变化有节奏（不是全部固定镜头）
- [ ] 特写/远景切换自然

---

## 阶段五：音频制作

### 5.1 配音
- [ ] 旁白配音已完成
- [ ] 对白配音已完成（每个角色独立配音）
- [ ] 配音情感与画面情绪匹配
- [ ] 配音语速合适（不太快/不太慢）
- [ ] 口型与画面基本同步

### 5.2 配乐
- [ ] BGM已选择/生成
- [ ] BGM风格与场景氛围匹配
- [ ] BGM音量合适（不盖过配音）
- [ ] 关键情绪点有BGM变化

### 5.3 音效
- [ ] 环境音效已添加（风声/雨声/城市噪音等）
- [ ] 动作音效已添加（脚步声/开门声/碰撞声等）
- [ ] 情绪音效已添加（心跳声/钟表声等）
- [ ] 音效与画面同步

---

## 阶段六：剪辑后期

### 6.1 粗剪
- [ ] 所有片段按分镜顺序排列
- [ ] 每个片段时长已调整
- [ ] 转场效果已添加
- [ ] 整体时长符合目标（____秒）

### 6.2 精剪
- [ ] 前3秒有强视觉冲击
- [ ] 节奏有张弛变化
- [ ] 对话节奏自然
- [ ] 无拖沓或过快的地方
- [ ] 结尾钩子效果强

### 6.3 字幕
- [ ] 对白字幕已添加
- [ ] 旁白字幕已添加（与对白样式区分）
- [ ] 字幕位置合适（不遮挡关键画面）
- [ ] 字幕时间与配音同步
- [ ] 无错别字

### 6.4 调色
- [ ] 全片色调统一
- [ ] 色调与题材匹配（玄幻偏冷/都市自然/古风暖色）
- [ ] 无过曝或过暗的画面
- [ ] AIGC标识已添加

### 6.5 导出
- [ ] 分辨率：1080P
- [ ] 帧率：30fps
- [ ] 格式：MP4 (H.264)
- [ ] 导出后已完整观看一遍

---
