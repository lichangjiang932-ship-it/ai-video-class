# ✅ 单集制作检查清单 — 金猫搭档

> 每集制作前逐项检查，确保质量。以一集90秒的金猫搭档播客视频为例。

---

## 阶段一：剧本检查

- [ ] 主题确定（本期播客话题）
- [ ] 豆包性格一致（热情、软乎乎、带紧张）
- [ ] 奶盖性格一致（冷淡、高冷、不耐烦）
- [ ] 对话有起承转合（开场→展开→吐槽→收尾）
- [ ] 笑点密度≥3个/90秒
- [ ] 每句对话≤20字（15秒内可读完）
- [ ] 脚本总字数≤600字（90秒×200字/分钟）

---

## 阶段二：分镜检查

- [ ] 每个镜头标注了时长（15秒/片段）
- [ ] 每个镜头标注了镜头类型（全景/特写/中景）
- [ ] 每个镜头标注了角色（豆包/奶盖/双人）
- [ ] 首尾帧衔接设计（尾帧→首帧相似）
- [ ] 正反打镜头设计（豆包↔奶盖切换）
- [ ] 全景镜头穿插（开场/转场/收尾）

---

## 阶段三：即梦生成检查

- [ ] 角色基准图已创建（豆包+奶盖各角度）
- [ ] 角色参考功能已开启
- [ ] 参考强度设为0.7-0.85
- [ ] 每个片段生成了4张/4段，选最佳
- [ ] 角色一致性检查通过
- [ ] 画面构图检查通过
- [ ] 毛发纹理检查通过

---

## 阶段四：首尾帧检查

- [ ] 每个15s片段的尾帧已提取
- [ ] 下一个片段的首帧与上一个尾帧相似
- [ ] 不相似的片段已用转场效果掩盖
- [ ] 转场时长0.3-0.5秒

---

## 阶段五：剪映剪辑检查

- [ ] 项目设置：9:16竖屏，1080×1920，30fps
- [ ] 视频片段按分镜顺序排列
- [ ] 调色参数已应用（亮度+8，对比度+12，饱和度+8，色温+10）
- [ ] 滤镜已应用（温暖，强度60%）
- [ ] 磨皮已应用（25%，锐化+8%）
- [ ] 柔光特效已应用（35%）
- [ ] 声波特效已添加（说话时出现）
- [ ] 字幕已添加（豆包白色，奶盖浅蓝色）
- [ ] 字幕位置：底部1/4处，字号24-28px
- [ ] 转场效果已添加（淡入淡出0.3秒）

---

## 阶段六：音频检查

- [ ] 豆包配音已添加（温暖男声，语速0.9倍）
- [ ] 奶盖配音已添加（冷淡女声，语速1.0倍）
- [ ] 音画同步检查通过
- [ ] BGM已添加（轻快风格，音量25%）
- [ ] 音效已添加（叮声、乌鸦声等，3-5个）
- [ ] 音量平衡：对话100%，BGM25%，音效60%

---

## 阶段七：导出发布检查

- [ ] 分辨率：1080p
- [ ] 帧率：30fps
- [ ] 格式：MP4
- [ ] 编码：H.264
- [ ] 画面无模糊/闪烁
- [ ] 音画无延迟
- [ ] 字幕无错别字
- [ ] 封面图已准备
- [ ] 标题已准备
- [ ] 标签已准备

---

## 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 角色不一致 | 重新生成，使用角色参考功能 |
| 毛发模糊 | 减少磨皮，增加锐化 |
| 配音不同步 | 逐帧调整配音位置 |
| BGM太响 | 降低BGM音量到20-30% |
| 色调不统一 | 使用"复制调节"统一色调 |
| 字幕遮挡角色 | 调整字幕位置到更下方 |
