## Step 1：选题与市场调研（30分钟）

### 1.1 用DeepSeek做热点分析

打开 chat.deepseek.com，开启联网搜索，输入：

```
请联网搜索2026年4月抖音漫剧的最新趋势：
1. 当前最火的都市逆袭类漫剧有哪些？
2. 这类漫剧的核心爽点模式是什么？
3. 有哪些常见套路已经被用烂了？
4. 用户评论中最常见的需求是什么？
请基于搜索结果，给我一个"都市逆袭"漫剧的差异化建议。
```

**DeepSeek返回的关键信息**：
- 都市逆袭是抖音漫剧第二大品类（仅次于玄幻）
- 常见套路："穷小子被看不起→获得金手指→打脸逆袭"已被用烂
- 用户反馈："想看更真实的逆袭，不要太离谱的金手指"
- 差异化方向：用"技术实力"作为逆袭手段，而非超能力

### 1.2 选题决策

基于调研结果，确定选题：

```
剧名：《代码之外》
核心设定：一个被裁员的程序员，凭借技术实力逆袭创业
差异化：没有金手指，纯靠技术和努力，更真实更有共鸣
核心爽点：技术碾压 + 都市逆袭 + 团队集结
目标受众：18-30岁男性，程序员/职场人
```

```
- 总集数：12集
- 单集时长：1分钟（约600-800字剧本）
- 目标受众：18-30岁男性
- 核心爽点：技术碾压+都市逆袭

【主角设定】
林晓，25岁，黑色短发，程序员，性格内向但倔强。

【请输出】

2. 5个核心角色设定（含外貌、性格、关系）
3. 12集分集大纲（每集一句话概要+核心爽点+钩子类型）
4. 爽点分布图（情绪强度1-10）
```

## Step 3：角色资产制作（30分钟）

### 3.1 生成主角基准图

打开即梦AI（jimeng.jianying.com），进入"文生图"：

**提示词**：
```
林晓，25岁男性，黑色短碎发，深棕色眼睛，棱角分明的面部轮廓，
穿着深灰色连帽卫衣和黑色牛仔裤，白色运动鞋，中等身材，
平静的表情，正面站立姿势，白色简洁背景，
日系都市动漫风格，高质量角色立绘，全身像
```

**参数设置**：
- 模型：通用v2.0
- 比例：9:16（竖屏）
- 生成数量：4张
- 采样步数：28步
- 引导强度：7.5

**操作**：点击"生成"，等待15-30秒，从4张中选择最佳的一张。

**选择标准**：
- ✅ 面部特征清晰
- ✅ 发型、发色明确
- ✅ 服装细节完整
- ✅ 姿态自然
- ❌ 避免手部遮挡面部的图

**下载保存**为 `角色资产/林晓-基准图-v1.png`

### 3.2 生成配角基准图

同样方法生成苏瑶的基准图：

**提示词**：
```
苏瑶，22岁女性，银白色及腰长发，蓝色眼睛，精致柔和的面部特征，
穿着白色连衣裙和浅蓝色针织开衫，苗条身材，
温柔的表情，正面站立姿势，白色简洁背景，
```

### 3.3 创建角色档案（Seedance 2.0）

## Step 7：音频制作（30分钟）

### 7.1 旁白配音

```
[疲惫]凌晨两点，林晓还在改一个他写了三天的bug。
[平静]第二天一早，他被叫进了老板的办公室。
```

### 7.2 对白配音

每个角色选择不同的音色：
- 林晓：年轻男声，低沉内敛
- 苏瑶：年轻女声，温柔清亮
- 老板：中年男声，冷漠威严
- 小胖：年轻男声，活泼

### 7.3 BGM选择

- 出租屋场景：轻柔的钢琴曲（孤独/疲惫）
- 老板办公室：低沉的弦乐（紧张/压抑）
- 天台场景：温暖的吉他曲（希望/转折）


### 8.3 精剪调整

- **前3秒**：出租屋全景+旁白，有视觉冲击（深夜+疲惫感）
- **节奏**：快-慢-快-慢交替
- **钩子**：结尾"一切开始改变"+两人剪影，留悬念
- **转场**：场景切换用淡入淡出（0.5秒）

### 8.4 字幕添加

- 使用剪映"智能字幕"自动识别
- 检查并修正错误
- 对白字幕：白色，底部居中
- 旁白字幕：斜体，底部居中，稍小

### 8.5 调色

- 整体色调偏暖（匹配都市感）
- 出租屋场景：偏暗偏冷（孤独感）
- 天台场景：偏暖偏金（希望感）

### 8.6 添加AIGC标识

在片尾添加文字："本视频由AI生成"

### 8.7 导出

- 分辨率：1080P
- 帧率：30fps
- 格式：MP4
- 导出后完整观看一遍确认无误

## 关键经验总结

### 1. 前期准备越充分，后期越省事
- 角色基准图做好了，后续所有镜头都能复用
- 场景母版图做好了，变体只需几分钟
- 剧本写好了，分镜和提示词都能直接推导

### 2. Draft模式是省钱利器
- 可灵AI的Draft模式先用低分辨率预览
- 确认效果后再渲染高清
- 每集能省60%的视频生成成本

### 3. 批量操作效率最高
- 同一工具的任务集中处理
- 提示词模板化，只改变量部分
- 4张候选中选最佳，减少重试

### 4. 质量检查不能省
- 每个环节完成后都要检查
- 最终成片必须完整看一遍
- 发现问题及时记录，下次改进
