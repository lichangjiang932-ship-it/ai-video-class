**124课时 · 17个模块 · 从零基础到独立制作完整AI漫剧**

> 本课程是一套完整的AI漫剧制作实战教程，覆盖从行业认知、剧本创作、AI图像/视频生成、音频制作、剪辑后期到平台分发变现的全流程。学完后你将具备独立制作高质量AI漫剧的能力。

**适用人群**：零基础入门者、短视频创作者、漫画从业者、影视从业者、创业者

**前置要求**：基本电脑操作能力，无需编程或美术基础

1. **先读学员手册**：[学员手册](AI漫剧制作课程/学员手册.md) 是课程导航地图，所有资料的入口
2. **选对学习路径**：根据你的时间和目标，选择速成/进阶/创业路径
3. **动手比看重要**：每学一个模块，立刻动手实操，产出驱动学习
4. **善用提示词模板**：[提示词模板库](AI漫剧制作课程/提示词模板库/场景分类提示词.md) 可以直接复制使用
