## 📖 课程简介

本课程是一套以"金猫搭档"宠物拟人漫剧为实战案例的全流程制作教程。你将学会用**即梦AI**生成电影质感的宠物拟人画面，用**剪映**完成后期拼接，制作出金毛犬"豆包"和布偶猫"奶盖"的播客对话漫剧——竖屏9:16，电影质感，每集15秒起。

整个课程只用两个核心工具：**即梦AI + 剪映**，零基础也能上手。

**适用人群**：宠物爱好者、短视频创作者、零基础入门者、想做IP的创业者

**前置要求**：基本电脑操作能力，无需编程或美术基础

---

## 🎯 课程特色

- **一个案例贯穿全程**：金毛犬"豆包"+布偶猫"奶盖"的播客对话漫剧
- **极简工具链**：即梦AI（生成）+ 剪映（后期），两个工具搞定全流程
- **电影质感输出**：竖屏9:16，光影氛围感拉满
- **可复制的工作流**：即梦AI生成15s视频→首尾帧提取→剪映拼接→成片

## 💡 为什么选择宠物拟人赛道？

1. **情感共鸣强**：宠物内容天然自带流量，用户粘性高
2. **制作门槛低**：宠物拟人比真人角色更容易保持一致性
3. **赛道蓝海**：AI宠物拟人漫剧竞争者少，先发优势明显
4. **变现路径清晰**：宠物周边、品牌联名、IP授权
5. **工具成熟**：即梦AI的宠物生成能力已非常稳定
