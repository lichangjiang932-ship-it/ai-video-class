## 📖 课程简介

本课程是一套完整的AI漫剧制作实战教程，覆盖从行业认知、剧本创作、AI图像/视频生成、音频制作、剪辑后期到平台分发变现的全流程。学完后你将具备独立制作高质量AI漫剧的能力。

**适用人群**：零基础入门者、短视频创作者、漫画从业者、影视从业者、创业者

**前置要求**：基本电脑操作能力，无需编程或美术基础

## 🚀 快速开始

### 方式一：直接阅读（推荐）

打开 **`全部课程-合并版.md`**，这是所有17个模块的完整合并内容，可以直接阅读或打印。

### 方式二：按模块学习

按学习路径依次阅读各模块文件：

**零基础速成（8周）**：
`模块01 → 02 → 03 → 04 → 06 → 10 → 11 → 12 → 17`

**专业进阶（16周）**：
`模块01 → 02 → 03 → ... → 17`（按顺序全部学习）

**创业导向（12周）**：
`模块01 → 02 → 04 → 06/07 → 09 → 10 → 11 → 14 → 15 → 16 → 17`

### 方式三：PPT教学

打开 **`AI漫剧制作全流程课程-2026版.pptx`**，42页精美幻灯片适合课堂教学或自学。详见 `讲师指南.md`。
