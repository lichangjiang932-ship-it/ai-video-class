# 🔒 安全策略

## 报告安全问题

如果你在课程内容中发现以下问题，请通过 [GitHub Issues](https://github.com/lichangjiang932-ship-it/ai-video-class/issues) 报告：

### 内容安全

- 包含过时或错误的工具链接（可能指向恶意网站）
- 包含可能导致用户账号风险的操作建议
- 包含违反平台规则的内容建议

### 版权问题

- 使用了未授权的版权素材
- 未正确标注素材来源

### 隐私问题

- 泄露了个人敏感信息
- 包含可识别个人身份的内容

## 报告格式

```
标题：[安全] 简要描述
内容：
- 问题类型：内容安全 / 版权 / 隐私
- 涉及文件：（文件名）
- 问题描述：（详细说明）
- 建议处理方式：（可选）
```

## 响应时间

- **严重问题**（恶意链接、隐私泄露）：24小时内处理
- **一般问题**（版权标注、过时信息）：7天内处理

## 免责声明

- 本课程中提到的所有第三方工具和服务，其安全性由各自平台负责
- 课程中的工具链接和价格信息可能随时变化，请以官方最新信息为准
- 使用课程中介绍的工具和方法时，请遵守相关法律法规和平台规则
